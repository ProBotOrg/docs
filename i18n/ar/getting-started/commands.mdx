# الاوامر

في هذا القسم ستحصل على شرح عميق لاوامر بروبوت و استخدامها و امثلة لاستخدام الاوامر.

:::important

- نظرا لان السلاش هو البريفكس الافتراضي للبوت فسيتم استخدامه بجميع الاستخدامات و الامثلة للاوامر. `/`.

- `[` `]` الاوامر المشار اليها بهذه العلامات هذا يعني انها اختيارية `(` `)`.

:::

## عامة 

| الامر | شرح | الاستخدامات |
| ------------- |:-------------:|:-------------:|
| /credits | يظهر رصيدك أو رصيد شخص اخر. | /credits [user] (amount) |
| /rep | منح شخص نقطة سمعة. يمكن استخدامها مرة واحدة فقط كل 24 ساعة. | /rep [user] |
| /moveme | ينقلك إلى روم صوتي اخر. | /moveme [channel or user] |
| /color | لتغيير لونك بالسيرفر. | /color [color's number] |
| /colors | يسرد لك كل الألوان المتاحة. | /colors |
| /short | اختصار رابط. | /short [URL] |
| /roll | رمي حجر نرد. | /roll (number) |

## اللفلات

| الامر | شرح | الاستخدامات |
| ------------- |:-------------:|:-------------:|
| /profile | عرض بطاقة التعريف الشخصية العامة المخصصة لك أو لشخص آخر. | /profile (user) |
| /rank | إظهار بطاقة الترتيب (الرانك) الخاصة بك او بشخص آخر بالسيرفر. | /rank (user) |
| /top | عرض الأعضاء الذين لديهم أكثر نقاط xp كتابي او صوتي في السيرفر. | /top (day/week/month) |
| /title | يغير عنوان بروفايلك. | /title [new title] |
| /setxp | تغيير اكسبي العضو. | /setxp [user] [type] [new_xp] |
| /setlevel | تغيير لفل العضو. | /setlevel [user] [type] [new_level] |

## معلومات

| الامر | شرح | الاستخدامات |
| ------------- |:-------------:|:-------------:|
| /user | يعرض معلومات، مثل تاريخ دخول السيرفر وتاريخ التسجيل في الديسكورد عنك او عن مستخدم آخر. | /user (user) |
| /avatar | الحصول على افتارك او افتار شخص اخر. | /avatar (user) |
| /server | يظهر معلومات حول السيرفر. | /server |
| /roles | احصل على قائمة برولات السيرفر وعدد الأعضاء. | /roles |

| /timeout | أعطاء وقت مستقطع للعضو. | /timeout [user] (time) (reason) |

| /lock | يمنع `@everyone` يقوم بتعطيل خاصية إرسال الرسائل من الجميع في روم محدد. | /lock (channel) (reason) |

| /setcolor | تغيير لون رول عن طريق الـHex Code. | /setcolor [role name] [color code]
| /slowmode | تفعيل او تعطيل سلو مود من الروم. | /slowmode (time) |
| /reset | اعادة تعيين النقاط الكتابيه/الصوتية/الانفايت من جميع الاعضاء او عضو معين. | /reset [text / voice / invites / limits) [all / user] |

<details>

<summary>ملاحظات ادارية مهمة</summary>

- يمكن للاعضاء الذين لديهم صلاحيات `ادمن ستريتر` او `رولات المشرفين` في لوحة التحكم استخدام جميع اوامر الاشراف على سبيل المثال يمكن للاعضاء الذين لديهم صلاحية الحظر باستخدام امر "/ban".

- اذا لم يتم تحديد وقت ستكون المدة الافتراضية ** ثلاثة اشهر** لجميع الاوامر التي تتطلب ادخال مدة معينة، يمكن تعديل المدة في اي وقت.
  
</details>

## البريميوم

:::caution 

الاوامر الاتية متاحة فقط لمشتركين البريميوم.

:::

| الامر | شرح | الاستخدامات |
| ------------- |:-------------:|:-------------:|
| /vip info | يعرض معلومات حول بوت بريميوم الخاص بك. | /vip |
| /vip move | لنقل بوت البريميوم لسيرفر اخر. `باقة 2` | /vip move [number] |
| /vip restart | لاعادة تشغيل بوت البريميوم. `باقة 2` | /vip restart |
| /vip whitelist | السماح للمستخدم بدخول لوحة التحكم. | /vip whitelist [user] |
| /vip avatar | لتغيير صورة البوت. `باقة 2` | /vip avatar |
| /vip username | لتغيير اسم البوت. `باقة 2` | /vip username [name] |
| /vip status | لتغيير حالة البوت. `باقة 2` | /vip status [status] |
| /vip transfer | تحويل ملكية البريميوم لمستخدم اخر. `باقة 2` | /vip transfer [user]
