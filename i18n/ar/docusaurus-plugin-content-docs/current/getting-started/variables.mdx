# متغيرات لوحة تحكم بروبوت

لإنشاء رسالة منسقة و إحترافية و مثالية يمكنك استخدام متغيرات مختلفة لفعل ذلك، سيوفر لك هذا المستند الشامل جميع المتغيرات التي يمكنك استخدامها في الأقسام المختلفة في لوحة التحكم.

## المتطلبات

قبل استخدام المتغيرات يجب عليك تفعيل وضع `Developer Mode` في جهازك وهذا سوف يسمح لك بالحصول على الايدي الذي تحتاجه لفعل اشياء محددة.

## تفعيل وضع الديفلوبر مود  

> على الكمبيوتر
1. قم بفتح برنامج الديسكورد
2. إنتقل إلى `User Settings`
3. ثم إذهب إلى `App Settings` وقم بالضغط على `Appearance`
4. الأن قم بتفعيل وضع `Developer Mode` 

> على الهاتف
1. قم بفتح الديسكورد
2. إنتقل إلى `User Settings`
3. ثم إذهب إلى `App Settings` وقم بالضغط على `Behavior`
4. الأن قم بتفعيل وضع `Developer Mode` 


بمجرد تفعيل وضع `Developer Mode` فأنت جاهز لاستخدام المتغيرات.

## متغيرات الترحيب

يمكنك استخدام المتغيرات الأتية في قسم الترحيب داخل لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[user]` | لمنشن العضو الجديد. | 
| `[inviter]` | لمنشن الداعي. |
| `[server]` | لذكر اسم السيرفر. |
| `[userName]` | لذكر اسم العضو الجديد بدون منشن. |
| `[inviterName]` | لذكر اسم الداعي بدون منشن. |
| `[memberCount]` |  لذكر عدد اعضاء السيرفر. |
| `<@user_id>` | لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` | لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` | لمنشن رول محدد باستخدام الايدي. |

## متغيرات الرد التلقائي

يمكنك استخدام المتغيرات الأتية في قسم الرد التلقائي داخل لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[user]` | لمنشن كاتب رسالة الرد. |
| `[userName]` |  لذكر اسم كاتب رسالة الرد بدون منشن. |
| `[invites]` |  لعرض عدد دعواتك. |
| `<@user_id>` |  لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` | لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` | لمنشن رول محدد باستخدام الايدي. |

## متغيرات رسائل الإيمبد

يمكنك استخدام المتغيرات الأتية في قسم رسائل الإيمبد داخل لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[text](link)` | هذا المتغير لربط نص بـ رابط محدد عند الضغط على النص ستنتقل الى الرابط. |
| `<@user_id>` |  لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` |  لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` |  لمنشن رول محدد باستخدام الايدي. |

## متغيرات نظام اللفلات 

يمكنك استخدام المتغيرات الأتية في قسم نظام اللفلات داخل لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[level]` |  لذكر اللفل الحالي للعضو. | 
| `[user] `| لمنشن العضو. |
| `[user.username]` | لذكر اسم العضو بدون منشن. |
| `[user.tag]` | لذكر تاق العضو. |
| `[oldLevel] ` | لذكر اللفل السابق للعضو. |

باستخدام المتغيرات المذكورة أعلاه سوف تجعل رسالتك تبدو مثالية و احترافية.
