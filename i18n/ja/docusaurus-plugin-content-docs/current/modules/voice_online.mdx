# ボイスオンライン

ユーザーがサーバーのボイスチャンネルにアクティブに接続しているメンバーの数を把握できる機能です。

:::caution

この機能は `Tier 2` のサブスクライバーのみ利用可能です。

:::

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## テキスト

部屋やカテゴリに表示・更新したいテキスト。[00]はメンバーカウントの変数で、ボットがボイスチャンネルにアクティブに接続しているユーザーの数を表示するために追加する必要があります。これにより、ユーザーはボイスチャンネル内のリアルタイムのメンバー数を確認でき、現在アクティブで参加している人数を簡単に把握できます。

## チャンネル

テキストを表示したいチャンネルを選択します。ボイスチャンネルまたはカテゴリのどちらでも構いません。この機能により、メンバーカウントテキストを表示するチャンネルを選択でき、サーバーに最適な場所を選ぶ柔軟性が提供されます。

# 隠しチャンネル

特定のチャンネルに接続しているメンバーをカウントから除外する機能です。これにより、そのチャンネルのメンバーはカウントに含まれません。

## Botを除外

これはオプションの設定で、有効にしない場合、接続しているボットの数もボイスオンラインシステムに含まれます。有効にすると、ボットの数は含まれません。この機能は、ボイスチャンネル内の人間のユーザー数を確認したいユーザーにとって便利です。

:::caution

メンバーカウントは**5分**ごとに更新されることに注意してください。

:::

全体として、ボイスオンラインはサーバーのボイスチャンネル内のメンバー数をリアルタイムで把握できる強力な機能であり、メンバーカウントテキストを表示する最適な場所を選ぶ柔軟性を提供します。また、特定のチャンネルをカウントから除外したり、ボットの数を総メンバー数から除外したりするオプションも提供され、サーバーの管理と監視に役立つ多用途なツールです。
