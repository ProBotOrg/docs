# ログ

ProBotがすべてのサーバーアクションを即座に記録します

:::info

ここで説明されている設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## 設定

各ログにはイベント名、ログチャンネル、および埋め込みカラーが含まれます。

:::important

一部のログはパフォーマンス上の理由から、ティア2のプレミアムストレージが必要になる場合があります。

:::

- メンバーBAN


- メンバーBAN解除


- タイムアウト（付与/解除）


- チャンネル作成


- チャンネル削除


- チャンネル更新


- チャンネル権限更新


- スレッドが作成されました


- スレッドが更新されました


- ロール作成


- ロール削除


- ロール更新


- メッセージ削除


- メッセージが編集されました


- メンバーをキック


- メンバーが別のボイスチャンネルに移動


- メンバーがボイスチャンネルから切断


- メンバー参加


- メンバー脱退しました


- ニックネーム変更


- 管理コマンド使用


- ロール付与


- ロール剥奪


- サーバー招待


- サーバー情報更新


- ボイスチャンネル参加


- ボイスチャンネル退出


- 音声状態 （マイク/スピーカーミュート）


- ボイスチャンネル切替


:::caution

時々、ボットが権限の問題でログにアクションを記録しないことがあります。その場合は、ボットをキックして再招待し、`Manage webhooks`の権限があることを確認してください。

:::
