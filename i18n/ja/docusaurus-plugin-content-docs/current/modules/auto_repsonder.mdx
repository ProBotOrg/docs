# 自動応答

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## セットアップ

応答を作成する前に、まず「応答を追加」ボタンをクリックする必要があります。これにより、自動応答の設定オプションが開き、トリガーと応答を設定できるようになります。

## トリガー

トリガーは、Botの応答を起動するキーフレーズやキーワードです。これは任意のもので構いません。また、ワイルドカードを設定するオプションもあり、完全な文や単一の単語の中にトリガーが含まれている場合でもBotが検出できるようになります。

## ワイルドカード

ボットがトリガーを検出できるようにします。完全な文や単一の単語の中にトリガーが含まれている場合でも検出可能です。

## 返信として送信

ボットの応答をトリガーに対してディスコードの返信オプションを使用して送信できます。これにより、トリガーを含むメッセージに直接返信し、応答が自動化されていることを明確にします。

## メッセージ送信者にメンション

返信オプションを通じてBotがユーザーにメンションを送ることもできます。これにより、ユーザーの名前がBotの応答に含まれ、返信に気づかせることができます。

## 応答

トリガーに対して受け取りたいテキストです。これはテキストボックスの下に表示されるユニークな変数を含むことができます。また、無制限のランダム応答を追加することができ、それぞれに応答テキストを設定できます。

## ランダム応答

無制限のランダム応答を追加することができ、それぞれに応答テキストを設定できます。これにより、ボットの応答がより多様で興味深いものになります。

:::caution

一度トリガーが設定されると、変更することはできません。そのため、トリガーを設定する際には混乱やエラーを避けるために慎重に行うことが重要です。

:::
