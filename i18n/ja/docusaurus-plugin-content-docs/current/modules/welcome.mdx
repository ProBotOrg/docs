# Discordサーバー管理のためのカスタムウェルカム＆グッバイメッセージ

ウェルカムメッセージとグッバイメッセージは、Discordサーバーでの温かく魅力的なコミュニティを築く上で重要な役割を果たします。これらのメッセージは、新しいメンバーに価値を感じさせ、帰属意識を提供します。同様に、メンバーがサーバーを離れるときに、よく作られたグッバイメッセージはポジティブな印象を与え、サーバーがメンバーの貢献を大切にしていることを示します。

このガイドでは、Discordサーバーでウェルカムメッセージとグッバイメッセージを設定するための手順を包括的に説明します。

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## ウェルカムメッセージの設定

### メッセージを送信する

ウェルカムメッセージは、新しいメンバーを迎え入れ、価値を感じさせる素晴らしい方法です。ウェルカムメッセージを設定するには、次の手順に従ってください：

1. ダッシュボードでウェルカムメッセージのオプションをオンにします。

2. サーバーに参加した新しいメンバーに送信したいテキストを入力します。

3. メッセージを新しいメンバーに直接送信するか、サーバーの特定のチャンネルに送信するかを選択します。

### 画像を送信する

ウェルカム画像を送信することも、新しいメンバーに温かく迎え入れる雰囲気を作り出す方法の一つです。ウェルカム画像を設定するには、次の手順に従ってください：

1. ダッシュボードでウェルカム画像のオプションをオンにします。

2. 画像を送信するための3つのオプションのいずれかを選択します：

- テキストメッセージと一緒に
- テキストメッセージの前に
- チャンネルに送信

利用可能なオプションを使用してウェルカム画像をカスタマイズします：

- **背景:** 透明な背景またはカスタマイズ可能な幅と高さの画像を選択します。
- **アバター:** 四角形または円形のアバターを選択し、高さ、幅、および座標をカスタマイズします。
- **ユーザー名:** ウェルカム画像上のユーザー名の位置、サイズ、色、および配置を選択します。
- **テキスト:** ウェルカム画像に表示したいテキストを入力し、その位置、サイズ、および色を設定します。

:::caution

背景画像は3MB未満である必要があり、サポートされている画像タイプは（`jpg, png, jpeg`）です。gifは現在サポートされていません。

:::

## 退出メッセージの設定

グッバイメッセージは、サーバーを離れたメンバーに感謝の意を示す素晴らしい方法です。グッバイメッセージを設定するには、次の手順に従ってください：

1. ダッシュボードでグッバイメッセージのオプションをオンにします。
2. サーバーを離れたメンバーに送信したいテキストを入力します。
3. 退出メッセージを送信したいチャンネルを選択します。

## 追加情報

:::info

- 各テキストメッセージボックスには、メッセージテキストに挿入できる変数があり、ユーザーのユーザー名やサーバーの名前などがあります。この機能を使用して、ウェルカムメッセージとグッバイメッセージをパーソナライズし、より魅力的にすることができます。

- ようこそメッセージと退出メッセージをカスタマイズすることは、サーバーの個性を示し、他のサーバーとの差別化を図る優れた方法です。さまざまなメッセージや画像を試して、コミュニティに最適なものを見つけてください。

:::
