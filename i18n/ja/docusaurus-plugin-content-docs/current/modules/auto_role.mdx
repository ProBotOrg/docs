# 自動ロール

ProBotの自動ロール機能を使用すると、Discordサーバーに新しく参加したユーザーに自動的にロールを割り当てることができます。このガイドでは、この機能を設定および構成する手順を説明します。

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

## 設定

自動ロールモジュールにアクセスしたら、新しいユーザーに自動的に割り当てられるロールを設定できます。これを行うには、モジュールの選択メニューを使用して、希望するロールを1つまたは複数選択します。

## プレミアム自動ロール

プレミアムサブスクライバーの場合、自動および特定のロール用にユニークなリンクを設定できます。

1. `追加`を選択します。
2. 次に、`編集`ボタンをクリックします。
3. 招待リンクを挿入します。
4. この招待リンクを通じて参加したメンバーに割り当てたいロールを選択します。

:::caution

1. Botのロールは選択した自動ロールよりも上位でなければなりません。
2. Discordのルール画面パネルが有効になっている場合、新しく参加したユーザーは自動ロールを受け取る前にルールを承認する必要があります。
3. Botは管理権限を持つロールをユーザーに割り当てません。

:::

これらの手順に従い、重要な注意点を考慮することで、ProBotを使用してDiscordサーバーで自動ロール機能を正常に設定および構成できます。
