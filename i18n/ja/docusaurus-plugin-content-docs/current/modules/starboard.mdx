# スターボードとは？

スターボードへようこそ！ここではメッセージが主役となります。この包括的なガイドでは、この素晴らしい機能の深淵を探り、サーバーの独自性に合わせてカスタマイズする方法を紹介します。スターボードの世界への航海に備えましょう。

## スターボードの解明

スターボードをサーバーの大舞台と想像してください。これは⭐️リアクションを通じてメッセージが注目を集める民主的なピンシステムです。メッセージが星のリアクション（例えば、3つ以上）を集めると、洗練されたボットが登場し、それを名誉ある`#starboard`チャンネルにエスコートします。魔法のように、どれだけ多くのメッセージがこの天体のスポットライトを浴びるかに制限はありません。これはサーバーの最も輝かしい瞬間を祝うものです。

## ダッシュボードのカスタマイズ

次に、ダッシュボードを通じてカスタマイズの核心に飛び込みましょう。これはサーバーの本質に調和するようにスターボードを作り上げるためのキャンバスです。ここにカスタマイズの宝石が待っています：

1. スターボード名: スターボードのキャッチーな名前を選びましょう。サーバーの個性を反映した記憶に残る名前にしてください。

2. スターボードチャンネル: スターボードをホストするチャンネルを決めましょう。スターのための専用スポットです！

3. 有効チャンネル: スターボード機能をどのチャンネルで有効にするかを選択できます。制限するか自由にするかはあなた次第です。

4. 有効ロール: スターボード機能をトリガーできるロールを選びましょう。限定するか全員に開放するかはあなた次第です。

5. スターの制限: メッセージがスターボードに載るために必要なスターの数を定義します。高く設定するか低く設定するかはあなたの判断です。

6. カスタム絵文字（プレミアム機能）: プレミアムユーザーであれば、スターのためのカスタム絵文字を選ぶこともできます。サーバーのスタイルを見せつけましょう！

7. スターボードメッセージ: スター付きメッセージをプレーンなメッセージとして表示するか、目を引く埋め込みとして表示するかを決めましょう。

8. 高度なオプション: もっとコントロールしたい方のために、高度なオプションには自己スターを無視する機能や`React to Starboard Post`機能の切り替えがあります。この機能は投稿のスター数を表示する便利なボタンを追加し、ユーザーが自分で管理できるようにします。

## グランドフィナーレ

結論として、スターボードは単にスターを集めるだけでなく、サーバーの瞬間をスタイリッシュに祝うものです。ミームをハイライトしたり、成果を記念したり、単に会話を楽しんだりする際に、スターボードはエレガントなタッチを加えます。

さあ、創造力を解き放ち、メッセージを輝かせましょう！スターボードで舞台を整えれば、星たちが続きます。⭐️✨
