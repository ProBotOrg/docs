# 一時的なチャンネル: ユーザーが管理できるProBotの機能

一時的なチャンネルは、ユーザーがサーバー上で自分の一時的なチャンネルを作成し管理できるProBotの素晴らしい機能です。この機能を使うことで、ユーザーは自分のチャンネルを自由に作成しカスタマイズできます。サーバーオーナーは、ユーザーのニーズに合わせてカスタムオプションを設定できます。このガイドでは、ProBotサーバーで一時的なチャンネルを設定する手順を説明します。カテゴリの作成やユーザーが利用できるオプションの設定方法についても説明します。

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。
:::

## 設定

### カテゴリの設定

まず最初に、一時的なチャンネル用の特別なカテゴリを作成する必要があります。ここにすべての一時的なチャンネルが配置されます。

### チャンネルの作成

この機能を使用するには、メンバーが指定されたボイスチャンネルに参加する必要があります。そのため、ボイスチャンネルを作成する必要があります。メンバーがそのチャンネルに参加すると、自分の一時的なチャンネルを作成できます。

:::info

サーバーのニーズに合わせて次のオプションを設定できます：

- **チャンネル削除時間**: 一時的なチャンネルが空のままで自動的に削除されるまでの時間です。このオプションはカスタマイズ可能で、任意の値に設定できます。

- **ユーザーごとの最大チャンネル数**: ユーザーが一度に作成できる一時的なチャンネルの最大数です。このオプションはチャンネルのスパムを防ぐために設定されており、最大数をカスタマイズできます。

- **チャンネルごとの最大ユーザー数**: 一時的なチャンネルに一度に参加できるユーザーの最大数です。このオプションはカスタマイズ可能で、最大ユーザー数を設定できます。

:::

:::caution

`メンバーを移動`の権限を持つユーザーは、ユーザー制限をバイパスします。

:::

これらのカスタマイズを行うことで、メンバーは自分の希望する設定で一時的なチャンネルを作成でき、管理も簡単になります。
