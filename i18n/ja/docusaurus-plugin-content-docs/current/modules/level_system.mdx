# レベルシステム

ProBotのレベリングシステムを使用すると、アクティブなサーバーメンバーは特定のレベルに達すると、自動的に特別なレベルロール、特権のある権限、ミステリーチャンネルへのアクセスが報酬として与えられます！

:::info

ここで説明する設定はダッシュボードで管理されます。

1. ダッシュボードにアクセスしてログインします。
2. 左側のメニューからサーバーを選択します。
3. モジュールを選択し、有効になっていることを確認します。

:::

最初に3つの異なるオプションがあり、それぞれを調整してより個別化されたレベリング体験を提供できます。

## XP対象外のロール

これを使用すると、選択したロールを持つ特定のユーザーがサーバー内のどこでも経験値を獲得するのを防ぐことができます。

## XP対象外のチャンネル

特定のチャンネルで経験値を獲得するのを防ぐことができます。

## ランクアップメッセージ

レベルアップしたメンバーにお祝いのメッセージを送信します。これらのメッセージにはユニークな変数も使用できます！

:::caution

**ランクアップメッセージ**は、テキストレベルでレベルアップしたユーザーにのみ送信されます。現在、ボイスレベルのメッセージはありません。

:::

## 設定

ロール報酬システムを確立する手順は以下の通りです：

`リワード獲得向けのロールを追加`というボタンがあり、そこからロール報酬、ユーザーが達成すべきレベル、および以下に詳述するその他のオプションを追加できます。

### テキストレベル

- メンバーが独立して設定された場合にレベルロールを取得するために達成する必要があるテキストレベルです。

- テキストレベルの範囲は0から100で、必要なレベルに達するごとに追加の報酬を提供します。

### ボイスチャットレベル

- メンバーが独立して設定された場合にレベルロールを取得するために達成する必要があるボイスレベルです。

- ボイスレベルの範囲は0から100で、必要なレベルに達するごとに追加の報酬を提供します。

### リワードを受けられるロール

メンバーがレベルを上げてあなたの要件を満たすと、特定のレベルでロールを付与できます。

:::caution

ロールが表示されない場合、Botのロールがすべてのサーバーロールの下にネストされています。この問題を解決するには、Botのロールをロール階層の一番上に移動して、ロールが表示されるようにします。

:::

## より高いリワードで削除する

ユーザーが高レベルに上がるときにロールを保持するか削除するかは設定で変更可能です。

- オプションを無効にすると、ロールはスタックします。

- オプションを有効にすると、高レベルに達するとロールが削除されます。

## より高いリワードで削除する

メンバーがテキストレベルのロールを受け取ったときにダイレクトメッセージで通知します。

## 重要な注意事項

:::important レベル要件を設定する際に発生する可能性のある3つの状況

1. **ボイス＆テキスト両方**: テキストとボイスの両方のレベルを設定し、どのオプションにも`0`を残さない場合、テキストとボイスの両方が必要になります。これにより、ユーザーはロール報酬を取得するために両方のレベルに達する必要があります。

2. **テキストのみ**: テキストレベルを設定し、ボイスを`0`にした場合、テキストのみが必要になります。これにより、ユーザーはロール報酬を取得するために必要なテキストレベルに達する必要があります。

3. **ボイスのみ**: ボイスレベルを設定し、テキストを`0`にした場合、ボイスのみが必要になります。これにより、ユーザーはロール報酬を取得するために必要なボイスレベルに達する必要があります。

4. **ロールポジション**: 多くのレベルに同じロール報酬を選択した場合でも心配ありません。指定したロールは、設定した任意のレベルの要件を満たすとユーザーに提供されます。

:::

:::caution

ユーザーがXPデータを更新し、ボイスレベルの報酬を受け取るためには、チャットで入力する必要があります。その後、ボットは報酬を与えることができます。

:::
