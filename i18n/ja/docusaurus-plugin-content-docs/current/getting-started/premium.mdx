# プレミアムBot情報

プレミアムBotの情報ページへようこそ。このページでは、プレミアムBotのプロセス、利用可能な機能、プレミアムティア、支払い方法、よくある質問について詳しく説明します。

## プレミアムティア

私たちのプレミアムプランは、サーバーを高度な機能とレイドや荒らしからの保護で強化したいサーバーオーナー向けに設計されています。プレミアムサービスは、それぞれ独自の機能を持つ2つのユニークなティアを提供します。

## プレミアムティア1

最初のティアは、手頃な価格で複数の機能を必要とする中規模サーバーに最適です。このティアに含まれる機能は次のとおりです：

- [保護](/modules/vip_protection.mdx)
- [一時リンク](/modules/temp_link.mdx) 
- [ボイスオンライン](/modules/voice_online.mdx)
- [特定の招待リンク用の自動ロール](/modules/auto_role.mdx)

:::caution

Botの設定を変更することはできず、招待リンクも編集できないことに注意してください。

:::

## プレミアムティア2

2番目のティアは、大規模サーバーに最適で、プレミアムティア1のすべての機能に加えて次の機能が含まれます：

- [アンチレイド](/modules/anti_raid.mdx)
- カスタムユーザー名、アバター、ステータス、専用コマンドを含むパーソナライズされたBot。

:::caution 

このティアではオプションのBotトークンを使用できないことに注意してください。

:::

## プレミアムBotの管理

専用ページからプレミアムBotを管理できます。各ティアの管理オプションは次のとおりです：

- **プレミアムティア1**: 支払いサイクルの編集、Botの招待、お好みの支払い方法での更新が可能です。

- **プレミアムティア2**: Botのプレゼンス、ユーザー名、アバター、ステータスの編集、Botの招待、ティア1と同様に更新が可能です。さらに、特別なニーズに対応するためのユニークなBot番号が付与されます。

## 支払いサイクルと価格

[プレミアム](https://probot.io/premium)ページで価格と支払いサイクルを確認できます。

:::tip 

__年間プランは50％割引になります__！

:::

## 支払い方法

次の方法のいずれかを使用してプレミアムメンバーシップを購入できます：

- **クレジットカード**
- **PayPal**
- **クレジット**

各支払い方法の詳細について：

- **__クレジットカード__**: 世界中の主要なクレジットカードをすべて受け付けています。表示される金額はドルです。

- **__PayPal__**: PayPalアカウントが正当で、支払い前に正しい金額があることを確認してください。問題が発生した場合は、クレジットカードをリンクしてみてください。

- **__クレジット__**: 私たちのサービスは`クレジット`というBot通貨を提供しています。実際のお金を使わずにさまざまな方法でクレジットをチャージできます。

## プレミアムBotの招待

プレミアム機能にアクセスするには、Botをサーバーに招待する必要があります。招待プロセスは、[プレミアム管理](https://probot.io/premium/manage)ページから完了できます。

:::caution 

Botを招待した後もプレミアム機能がロックされたままの場合は、プレミアムBotまたはプレミアムBotと標準Botの両方をキックしてから、プレミアムBotを再度サーバーに招待してみてください。プレミアムバージョンを購入した後、標準Botを削除できます。

:::

## よくある質問

<details>
<summary>返金をリクエストできますか？</summary>

はい、できます。ただし、返金ポリシーを確認し、購入日から7日以内に合理的な理由で返金をリクエストする必要があります。

</details>

<details>

<summary>同じサーバーで複数のサブスクリプションを持つことはできますか？</summary>

いいえ、サーバーごとに1つのプレミアムBotしか持てません。

</details>

<details>

<summary>プレミアムBotの所有権を友人に譲渡できますか？</summary>

はい、できます！ただし、これは`ティア2`プランに限定されます。

</details>

<details>

<summary>自動更新をキャンセルできますか？</summary>

はい、できます！プレミアム管理ページからキャンセルをクリックしてください。

</details>
