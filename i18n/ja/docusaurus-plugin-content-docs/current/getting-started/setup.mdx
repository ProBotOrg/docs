# ProBotのセットアップ

:::info

ProBotをサーバーに招待するには、特定の権限が必要です。これらの権限がないと、Botをサーバーに追加することはできません。必要な権限は以下の通りです：

- `管理者` 権限
- `サーバー管理` 権限

:::

## ステップ1: ProBotを招待する

[招待リンク](https://probot.io/invite)を使用して、ProBotをサーバーに招待し、アカウントを認証してサーバーを選択します。

## ステップ2: 認証と権限の付与

ProBotを招待すると、アカウントの認証とBotが正常に機能するために必要な権限の選択を求められます。

## ステップ3: ProBotのロールを設定する

ダッシュボードの一部のモジュールでサーバーのロールを表示するには、ProBotのロールをサーバーのロール階層の一番上に移動します。これを行うには：
- サーバー設定に移動
- ロールをクリック
- ProBotのロールをクリックしてドラッグし、リストの一番上に移動

:::caution

ProBotのロールを一番上に移動しても、Discordがダッシュボードにロールを表示しない場合があります。その場合は、次の手順を試してください：
1. ProBotのロールを下にドラッグ
2. 変更を保存
3. ProBotのロールを上にドラッグ
4. 再度変更を保存

:::

## サポートが必要ですか？

:::info

ProBotのセットアップにさらにサポートが必要な場合は、[サポートサーバー](https://discord.gg/probot)に参加して助けを求めてください。私たちのチームが喜んでお手伝いします！

:::
