# ダッシュボード: サーバー管理のワンストップショップ

ダッシュボードはサーバー設定の中心です。すべてのモジュールとコマンドを管理するための使いやすいインターフェースを提供します。複雑で時間のかかるコマンドライン設定にさようなら！

## ダッシュボードへのアクセス

ダッシュボードへのアクセスは簡単でわかりやすいです。以下の手順に従ってください：
- [ウェブサイト](https://probot.io/)にアクセスし、アカウントにログインします。
- 同じページにダッシュボードボタンが表示されるので、それをクリックしてダッシュボードにアクセスします。

:::caution
サーバーをリストに表示するには「管理者」権限が必要です。
:::

## モジュールとコマンドの設定

ダッシュボードの各モジュールには右上隅にスイッチボタンがあり、そのモジュール内のすべてのコマンドを無効にすることができます。さらに、各コマンドには有効/無効にするためのスイッチボタンとカスタマイズするための編集ボタンがあります。

各コマンドで利用可能なカスタマイズオプションは次のとおりです：
- **エイリアス**: コマンドのカスタムエイリアスを設定し、覚えやすく使いやすくします。
- **有効なロール**: 特定の役割にコマンドの使用を制限します。
- **無効なロール**: 特定の役割がコマンドを使用できないようにします。
- **有効なチャンネル**: 指定されたチャンネルでのみボットがコマンドに応答します。
- **無効なチャンネル**: これらのチャンネルではボットがコマンドに応答しません。
- **メッセージ削除時の自動削除**: 送信者がメッセージを削除した場合、ボットの応答も削除されます。
- **コマンド呼び出しメッセージの自動削除**: 送信者が送信したコマンドメッセージは送信直後に削除されます。
- **ボットの返信メッセージを5秒後に自動削除**: ボットの返信は送信後5秒で削除されます。

:::caution
サーバーがリストに表示されない場合は、ダッシュボードへの完全なアクセスに必要な「管理者」権限がないことを意味します。役割設定で「管理者」権限が明示的に付与されていることを確認してください。
:::
