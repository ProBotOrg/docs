# メンバーシップ

メンバーシップ機能は、非会員には利用できない独占的な特典と特権をユーザーに提供します。ユーザーは通常、これらの特典にアクセスするために月額または年額の料金を支払う必要があります。

利用可能なメンバーシップの階層は3つあります：

- **シルバー**
- **ゴールド**
- **ダイヤモンド**

各階層は異なる特典を提供し、異なる予算と使用要件を持つユーザーのニーズに応えるように設計されています。

## シルバーメンバーシップ階層

シルバーメンバーシップ階層はエントリーレベルの階層で、以下の特典にアクセスできます：

- 毎日(x3)、最大10,000クレジット
- 1日5回の評価ポイント
- カスタマイズ可能な背景（プロフィール / ID）

:::note
このメンバーシップ階層は、比較的控えめなニーズを持ち、メンバーシップの一部の特典をコスト効率よく利用したいユーザーに最適です。
:::

## ゴールドメンバーシップ階層

ゴールドメンバーシップ階層は中間レベルの階層で、以下の特典にアクセスできます：

- 毎日(x5)、最大25,000クレジット
- 1日10回の評価ポイント
- カスタマイズ可能な背景（プロフィール / ID）
- 月に1回のアカウント転送

:::note
このメンバーシップ階層は、シルバー階層では提供できないより高いレベルの機能を必要とする、より大きなニーズを持つユーザー向けに設計されています。
:::

## ダイヤモンドメンバーシップ階層

ダイヤモンドメンバーシップ階層は最上位の階層で、以下の特典にアクセスできます：

- 毎日(x10)、最大50,000クレジット
- 1日30回の評価ポイント
- カスタマイズ可能な背景（プロフィール / ID）
- 月に1回のアカウント転送

:::note
このメンバーシップ階層は、高いレベルの機能と独占的な特典へのアクセスを必要とするパワーユーザーに最適です。
:::

## Discordでのデイリーリワードの授与

Discordプラットフォーム上のデイリーコマンドにより、ユーザーは毎日クレジットを受け取ることができます。授与されるクレジットの量は、各メンバーシップ階層の説明に記載されているように、ユーザーのメンバーシップ階層によって異なります。

## デイリー評価ポイント

デイリー評価ポイント機能により、ユーザーは`/rep`コマンドを使用して友達に評価ポイントを与えることができます。これらのポイントは友達のプロフィールに表示されます。

## 背景の変更

背景の変更機能により、ユーザーはプロフィールとIDのために独自の背景画像をアップロードするオプションを提供します。

:::note
サポートされているファイルタイプは`png`、`jpeg`、および`SVG`に限定されており、画像のサイズは3MBを超えないようにする必要があります。
:::

:::warning
不適切な背景やコンテンツ、`+18`および`NSFW`コンテンツは厳しく禁止されています。これらのガイドラインに違反すると、ユーザーのサブスクリプションが削除され、再度メンバーシップを購入する権限が失われる可能性があります。
:::

## 月に1回の転送

月に1回の転送機能により、ユーザーはレベルやクレジットを含む古いアカウントデータを月に1回転送することができます。このプロセスを完了するには、ユーザーは古いアカウントIDとメールアドレスを提供する必要があります。提供されたメールアドレスは、古いアカウントに関連付けられているメールアドレスと一致する必要があります。ユーザーは転送プロセスを完了するために、メールで6桁の認証コードを受け取ります。

:::note
プロフィールデータを転送する機能は月に1回のみ利用可能です。この制限により、転送プロセスが公平であり、機能の乱用を防ぐことができます。
:::

:::note
認証コードは15分間のみ有効であることを覚えておくことが重要です。ユーザーがその時間内にコードを受け取らない場合、メールとスパムフォルダーを確認するか、別のコードを待つ必要があります。
:::

## メンバーシップの価格

シルバー、ゴールド、およびダイヤモンドメンバーシップ階層の価格を確認するには、当社のウェブサイトの[メンバーシップページ](https://probot.io/memberships)をご覧ください。そこで、各階層の特典を比較し、ニーズと予算に最も適したものを選択できます。さらに質問や懸念がある場合は、[カスタマーサポート](https://discord.gg/probot)チームにお問い合わせください。
