# はじめに

ProBotは、高度にカスタマイズ可能な多機能Botで、画像付きのウェルカムメッセージ、詳細なログ、便利なコマンド、モデレーション、リアクションロール、SNS通知、サーバー保護などの機能を提供します。

## ドキュメントについて

当社のドキュメントでは、ProBotの各機能の使用方法と設定方法、そして問題が発生した場合のトラブルシューティングのヒントについて包括的に説明しています。

## モジュール

左側のサイドバーにあるモジュール名をクリックすると、簡単に特定のモジュールに移動できます。各モジュールには独自の設定ページがあります。

## コマンド

すべてのProBotコマンドには、使用例と情報が記載された専用ページがあります。詳しくは[コマンドページ](https://probot.io/commands)をご覧ください。

## フォーマット

このドキュメントでは、以下のカラ―コードシステムを使用しています：

:::tip
機能に関するヒントを示します。
:::

:::info
機能に関する情報を提供します。
:::

:::caution
機能に関する注意点を強調します。
:::

:::note
機能に関するメモを表示します。
:::

:::danger
既知のバグや問題、ワークフローに悪影響を与える可能性のある事項を示します。
:::
