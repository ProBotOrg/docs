# Mensajes de bienvenida y despedida personalizados para una gestión atractiva de servidores de Discord

Los mensajes de bienvenida y despedida tienen un papel fundamental en establecer una comunidad acogedora y atractiva en tu servidor de Discord. Estos mensajes pueden hacer que un nuevo miembro se sienta valorado y proporcione un sentido de pertenencia. De manera similar, cuando un miembro abandona el servidor, un mensaje de despedida bien elaborado puede crear una impresión positiva y mostrar que el servidor valora la contribución del miembro.

Esta guía proporciona una descripción general completa de los pasos necesarios para configurar mensajes de bienvenida y despedida en tu servidor de Discord.

:::info

El dashboard es donde se gestionan las configuraciones descritas aquí.

1. Accede al dashboard e inicia sesión.
2. Selecciona tu servidor en el menú de la izquierda.
3. Elige el apartado y asegúrate de que esté habilitado.

:::

## Configuración de Mensajes de Bienvenida

### Envío de un Mensaje

Un mensaje de bienvenida es una excelente manera de dar la bienvenida a nuevos miembros y hacer que se sientan valorados. Para configurar mensajes de bienvenida, sigue estos pasos:

1. Activa la opción para los mensajes de bienvenida en el dashboard.

2. Ingresa el texto que deseas enviar a los nuevos miembros cuando se unan al servidor.

3. Elige si deseas enviar el mensaje como un mensaje directo al nuevo miembro o a un canal específico en el servidor.

### Envío de una Imagen

Enviar una imagen de bienvenida es otra forma de crear una atmósfera cálida y acogedora para los nuevos miembros. Para configurar imágenes de bienvenida, sigue estos pasos:

1. Activa la opción para las imágenes de bienvenida en el dashboard.

2. Elije una de las tres opciones disponibles para enviar la imagen:

- Con mensaje de texto
- Antes del mensaje de texto
- Enviar a un canal

Personaliza la imagen de bienvenida utilizando las opciones disponibles:

- **Fondo:** Elige entre un fondo transparente o una imagen con ancho y alto personalizables.
- **Avatar:** Elige entre un avatar cuadrado o circular con altura, ancho y coordenadas personalizables.
- **Nombre de usuario:** Elige la posición, tamaño, color y alineación del nombre de usuario del usuario en la imagen de bienvenida.
- **Texto:** Ingresa el texto que deseas mostrar en la imagen de bienvenida, junto con su posición, tamaño y color.

:::caution

Las imágenes de fondo deben ser inferiores a 3MB, y los tipos de imagen admitidos son (`jpg, png y jpeg`). Los archivos gif no son compatibles actualmente.

:::

## Configuración de Mensajes de Despedida

Un mensaje de despedida es una excelente manera de mostrar tu apreciación a los miembros que han dejado tu servidor. Para configurar mensajes de despedida, sigue estos pasos:

1. Activa la opción para los mensajes de despedida en el dashboard.
2. Ingresa el texto que deseas enviar a los miembros cuando abandonen el servidor.
3. Elige el canal donde deseas que se envíe el mensaje de despedida.

## Información Adicional

:::info

- Cada cuadro de mensaje de texto tiene una variable utilizable que se puede insertar en el texto del mensaje, como el nombre de usuario del usuario o el nombre del servidor. Esta característica puede ayudar a personalizar los mensajes de bienvenida y despedida y hacerlos más atractivos.

- Personalizar los mensajes de bienvenida y despedida es una excelente manera de mostrar la personalidad de tu servidor y hacer que destaque entre otros servidores. Considera experimentar con diferentes mensajes e imágenes para encontrar lo que funciona mejor para tu comunidad.

:::
