# Auto-Roles

La función de Auto-Roles de ProBot permite asignar automáticamente roles a los usuarios recién unidos en un servidor de Discord. Esta guía te guiará a través de los pasos para configurar y configurar esta función.

:::info

El dashboard es donde se gestionan las configuraciones descritas aquí.

1. Ve al dashboard e inicia sesión.
2. Selecciona tu servidor en el menú de la izquierda.
3. Elije el apartado y asegúrate de que esté habilitado.

:::

## Configuración

Una vez que hayas accedido al apartado de Auto-Roles, puedes comenzar a configurar los roles que se asignarán automáticamente a los nuevos usuarios. Para hacer esto, utiliza el menú de selección del apartado para elegir uno o varios roles según lo deseado.

## Auto-Role Premium

Si eres un suscriptor premium, puedes establecer un enlace único para auto-roles particulares.

1. Selecciona `Agregar`
2. Luego, haz clic en el botón `Editar`.
3. Inserta el enlace de invitación.
4. Selecciona los roles que te gustaría que los miembros tengan si se unen a través de este enlace de invitación.

:::caution

1. El rol del bot debe ser superior al auto-rol elegido.
2. Si el panel de pantalla de reglas de Discord está habilitado, los usuarios recién unidos deberán aceptar las reglas antes de recibir el auto-rol.
3. El bot no asignará roles a los usuarios con privilegios administrativos.

:::

Siguiendo estos pasos y tomando nota de las notas importantes, puedes configurar con éxito la función de Auto-Roles en tu servidor de Discord utilizando ProBot.
