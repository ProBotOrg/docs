# Logs

Registra todas las acciones en el servidor con ProBot.

:::info

La configuración se administra en el dashboard.

1. Accede al dashboard y inicia sesión.
2. Selecciona tu servidor del menú de la izquierda.
3. Elige el apartado y asegúrate de que está habilitado.

:::

## Configuraciones

Cada registro contendrá un interruptor de alternancia, un canal de registro y un color de embed.

:::important

Algunos registros pueden requerir premium de nivel 2 por razones de rendimiento.

:::

- Usuario expulsado

- Usuario temporalmente suspendido

- Canal creado

- Rol creado

- Canal eliminado

- Mensaje eliminado

- Rol eliminado

- Mensaje editado

- Usuario expulsado

- Usuario movido a otro canal de voz

- Usuario desconectado del canal de voz

- Usuario unido

- Usuario abandonó

- Apodo cambiado

- Comando de moderación utilizado

- Rol asignado

- Rol eliminado

- Invitaciones de servidores

- Usuario desbaneado

- Canal actualizado

- Permisos del canal actualizados

- Rol actualizado

- Actualización del servidor

- Usuario unido al canal de voz

- Usuario abandonó el canal de voz

- Estado de voz

- Usuario cambió entre canales de voz

:::caution

Ocasionalmente, el bot puede no registrar acciones en los registros debido a un problema de permisos; si esto ocurre, expulsa y vuelve a invitar al bot y asegúrate de que tenga permisos de `Gestionar webhooks`.

:::
