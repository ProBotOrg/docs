# ڕێکخستنی پرۆبۆت

:::info
بۆ بانگهێشتکردنی **پرۆبۆت** بۆ سێرڤەرەکەت، پێویستە هەندێک مۆڵەتت هەبێت.  
بەبێ ئەم مۆڵەتانە ناتوانیت پرۆبۆت زیاد بکەیت بۆ سێرڤەرەکەت.

مۆڵەتە پێویستەکان:

- مۆڵەتی `Administrator`
- مۆڵەتی `Manage Server`
:::

---

## هەنگاوی یەکەم: بانگهێشتکردنی پرۆبۆت

بۆ بانگهێشتکردنی پرۆبۆت بۆ سێرڤەرەکەت،  
ئەم [لینکی بانگهێشت](https://probot.io/invite) ـە بەکاربهێنە.

پاش ئەوە:
- هەژمارەکەت پەیوەست بکە
- سێرڤەرەکەت هەڵبژێرە بۆ بانگهێشتکردنی پرۆبۆت

---

## هەنگاوی دووەم: پەیوەستکردن و دانانی مۆڵەتەکان

کاتێک پرۆبۆت بانگهێشت دەکەیت،  
داوای پەیوەستکردنی هەژمارەکەت و دانانی مۆڵەتە پێویستەکان دەکرێت  
بۆ ئەوەی بۆت بتوانێت بە شێوەی دروست کار بکات.

دڵنیابە کە هەموو مۆڵەتە پێویستەکان قبوڵ دەکەیت.

---

## هەنگاوی سێیەم: ڕێکخستنی ڕۆڵەکان بۆ پرۆبۆت

بۆ ئەوەی ڕۆڵەکان لە هەندێک بەشی داشبۆرددا پیشان بدرێن،  
پێویستە ڕۆڵی **پرۆبۆت** لە سەرەوەی هەموو ڕۆڵەکانی تر بێت.

بۆ ئەنجامدانی ئەمە:

- بچۆ بۆ **Server Settings**
- بچۆ ناو بەشی **Roles**
- ڕۆڵی پرۆبۆت بگرە و ڕابکێشە بۆ سەرەوەی لیستەکە

---

:::caution
**تێبینی:**  
هەندێک جار دیسکۆرد ڕۆڵەکان لە داشبۆرددا پیشان نادات،  
تەنانەت ئەگەر ڕۆڵی پرۆبۆت لە سەرەوە بێت.

ئەگەر ئەم کێشەیە ڕوویدا، ئەم هەنگاوانە تاقی بکەوە:

1. ڕۆڵی پرۆبۆت ڕابکێشە خوارەوە
2. گۆڕانکارییەکان هەڵبگرە
3. دووبارە ڕۆڵی پرۆبۆت ڕابکێشە بۆ سەرەوە و گۆڕانکارییەکان هەڵبگرە
:::

---

## پێویستت بە یارمەتیدان هەیە؟

:::info
ئەگەر پێویستت بە یارمەتیدانی زیاتر هەیە بۆ ڕێکخستنی **پرۆبۆت**،  
دەتوانیت بچیتە ناو  
[سێرڤەری پشتگیری فەرمی](https://discord.gg/probot)  
و داوای یارمەتیدان بکەیت.

تیمی ئێمە بە دڵخۆشییەوە یارمەتیت دەدات ❤️
:::
