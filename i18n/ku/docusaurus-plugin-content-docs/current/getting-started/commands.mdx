# فەرمانەکان

لەو بەشەدا، تێگەیشتنێکی تەواو دەست دەهێنیت لە فەرمانەکانی **ProBot**، چۆنیەتی بەکارهێنانیان و نموونەی جێبەجێکردنیان.

:::important

- چونکە فەرمانەکانی Slash بنەڕەتی فەرمانی بۆتەکەن، هەموو بەکارهێنان و نموونەکان بە `/` پیشاندەدرێن.
- نیشانەکانی `[` `]` بۆ هەر فەرمانێک پێویستن، بەڵام `(` `)` هەڵبژاردەیین (اختیاری).

:::

## گشتی

| فەرمان | ڕوونکردنەوە | بەکارهێنان |
| ------------- |:-------------:|:-------------:|
| /credits | پیشاندانی باڵانسی خۆت یان باڵانسی کەسێکی تر | /credits [user] (amount) |
| /rep | دانانی خاڵی ڕەزامەندی بۆ کەسێک (پێویستی بە Premium هەیە) | /rep [user] |
| /moveme | گواستنەوەی خۆت بۆ ژوورێکی دەنگی | /moveme [channel or user] |
| /color | گۆڕینی ڕەنگی ڕۆڵەکەت لە سێرڤەر | /color [color number] |
| /colors | پیشاندانی هەموو ڕەنگە بەردەستەکان | /colors |
| /short | کورتکردنەوەی لینک | /short [URL] |
| /roll | فڕێدانی تاس بۆ ژمارەیەکی هەڕەمه‌کی | /roll (number) |

## لێڤڵ (XP)

| فەرمان | ڕوونکردنەوە | بەکارهێنان |
| ------------- |:-------------:|:-------------:|
| /profile | پیشاندانی پڕۆفایلەکەت یان کەسێکی تر | /profile (user) |
| /rank | پیشاندانی کارتی پلە (Rank) لە سێرڤەر | /rank (user) |
| /top | پیشاندانی ئەندامانی زۆرترین XP | /top (day/week/month) |
| /title | گۆڕینی ناونیشانی پڕۆفایل | /title [new title] |
| /setxp | گۆڕینی XP ـی ئەندام | /setxp [user] [type] [new_xp] |
| /setlevel | گۆڕینی ئاستی (Level) ئەندام | /setlevel [user] [type] [new_level] |

## زانیاری

| فەرمان | ڕوونکردنەوە | بەکارهێنان |
| ------------- |:-------------:|:-------------:|
| /user | پیشاندانی زانیاری وەک بەرواری چوونەژوور و تۆماربوون | /user (user) |
| /avatar | وەرگرتنی وێنەی Avatar | /avatar (user) |
| /server | زانیاری گشتی دەربارەی سێرڤەر | /server |
| /roles | لیستی ڕۆڵەکان و ژمارەی ئەندامانیان | /roles |

| /reset | ڕێستکردنی XP / Voice / Invites | /reset [type] [all/user] |

<details>
<summary>تێبینییە گرنگەکان بۆ بەڕێوەبەران</summary>

- تەنها ئەو بەکارهێنەرانەی ڕێگەپێدانی **Administrator** یان **Mod Roles** ـیان هەیە دەتوانن ئەم فەرمانانە بەکاربهێنن.
- ئەگەر کات دیاری نەکرابێت، کاتی بنەڕەتی بۆ فەرمانەکان **٣ مانگ** ـە.

</details>
