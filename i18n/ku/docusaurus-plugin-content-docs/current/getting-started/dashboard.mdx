# داشبۆرد: بنەمای تەواوی بەڕێوەبردنی سێرڤەرەکەت

داشبۆرد بنەمایەکی سەرەکییە بۆ ڕێکخستنی سێرڤەرەکەت، ڕووکارییەکی سادە و بەکارهێنانی ئاسان پێشکەش دەکات بۆ بەڕێوەبردنی هەموو بەشەکان و فەرمانەکان.  
دوورخستنەوە لە فەرمانە ئاڵۆز و کات‌کوژ! 🚀

## چوونەژوور بۆ داشبۆرد

چوونەژوور بۆ داشبۆرد زۆر سادە و ڕاستەوخۆیە، ئەم هەنگاوانە جێبەجێ بکە:

- بچۆ بۆ [ماڵپەڕی ئێمە](https://probot.io/) و بچۆ ناو هەژمارەکەت.
- لە هەمان پەڕەدا دوگمەی **داشبۆرد** دەبینیت، کرتەی لێ بکە بۆ چوونەژوور.

:::caution
پێویستە مۆڵەتی `Administrator` ـت هەبێت بۆ ئەوەی سێرڤەرەکەت لە لیستەکەدا دەربکەوێت.
:::

## ڕێکخستنی بەشەکان و فەرمانەکان

لە هەموو بەشەکانی داشبۆرد، دوگمەیەک هەیە لە گووشەی سەرەوە ـی چەپ کە ڕێگەت پێدەدات هەموو فەرمانەکان لەو بەشەدا ناچالاک بکەیت.

هەروەها:
- دوگمەیەک لە تەنیشت هەر فەرمانێکدا هەیە بۆ چالاک / ناچالاککردنی فەرمانەکە
- دوگمەی **دەستکاری (Edit)** بۆ ڕێکخستنی تایبەتمەندییەکانی فەرمان

### هەندێک هەڵبژاردەی ڕێکخستن بۆ هەر فەرمانێک

- **کورتکراوەکان (Aliases)**: کورتکراوەی تایبەتی بۆ فەرمان بۆ ئاسانتر بەکارهێنان.
- **ڕۆڵە چالاکەکان**: تەنها ئەو ڕۆڵانە دەتوانن فەرمانەکە بەکاربهێنن.
- **ڕۆڵە ناچالاکەکان**: قەدەغەکردنی فەرمان بۆ ڕۆڵە دیاریکراوەکان.
- **ڕۆومە چالاکەکان**: فەرمان تەنها لە ژوورە دیاریکراوەکاندا کار دەکات.
- **ڕۆومە ناچالاکەکان**: فەرمان لە ژوورە دیاریکراوەکاندا کار ناکات.
- **سڕینەوەی وەڵامی بۆت بە شێوەی خۆکار**: ئەگەر پەیامی ئەندام بسڕدرێتەوە، وەڵامی بۆتیش دەسڕدرێتەوە.
- **سڕینەوەی خۆکاری پەیامی فەرمان**: پەیامی فەرمان دوای ناردن خۆکارانە دەسڕدرێتەوە.
- **سڕینەوەی وەڵامی بۆت دوای ٥ چرکە**: وەڵامی بۆت دوای 5 چرکە خۆکارانە دەسڕدرێتەوە.

:::caution
سێرڤەرەکەت لە لیستەکەدا نادیارە؟  
ئەمە واتە مۆڵەتی `Administrator` ـت نییە.  
تکایە دڵنیابە لەوەی مۆڵەتی `Administrator` ـت پێدراوە.
:::
