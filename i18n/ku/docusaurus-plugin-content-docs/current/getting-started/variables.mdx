# گۆڕاوەکانی داشبۆردی پرۆبۆت

بۆ دروستکردنی پەیامێکی ڕێکخراو، جوان و پیشەیی،  
دەتوانیت گۆڕاوە (Variables) ـی جیاواز بەکاربهێنیت.  

ئەم بەڵگەنامەیە هەموو ئەو گۆڕاوەی پێشکەش دەکات  
کە دەتوانیت لە بەشە جیاوازەکانی داشبۆردی پرۆبۆت بەکاریان بهێنیت.

:::caution
پێش بەکارهێنانی گۆڕاوەکان، پێویستە **Developer Mode** چالاک بکەیت.  
ئەم دۆخە ڕێگەت پێدەدات ناسنامە(ID) ـە پێویستەکان وەربگریت بۆ هەندێک کردار.
:::

---

## چالاککردنی Developer Mode

### لەسەر کۆمپیوتەر

1. ئەپی دیسکۆرد بکەوە  
2. بچۆ بۆ `User Settings`
3. لەژێر `App Settings` بچۆ بۆ `Appearance`
4. **Developer Mode** چالاک بکە

### لەسەر مۆبایل

1. دیسکۆرد بکەوە
2. بچۆ بۆ `User Settings`
3. لەژێر `App Settings` بچۆ بۆ `Behavior`
4. **Developer Mode** چالاک بکە

:::note
کاتێک Developer Mode چالاک کرا، ئێستا ئامادەیت بۆ بەکارهێنانی گۆڕاوەکان.
:::

---

## گۆڕاوەکانی پێشواز (Welcome)

ئەم گۆڕاوانە دەتوانرێت لە بەشی پێشواز لە داشبۆرددا بەکاربهێنرێن:

| گۆڕاو | ڕوونکردنەوە |
| ------------- |:-------------:|
| `[user]` | منشن‌کردنی ئەندامی نوێ |
| `[inviter]` | منشن‌کردنی بانگهێنەر |
| `[server]` | ناوی سێرڤەر |
| `[userName]` | ناوی ئەندام (بێ منشن) |
| `[inviterName]` | ناوی بانگهێنەر (بێ منشن) |
| `[memberCount]` | ژمارەی ئەندامانی سێرڤەر |
| `<@user_id>` | منشن‌کردنی ئەندام بە ID |
| `<#channel_id>` | منشن‌کردنی ژوور بە ID |
| `<@&role_id>` | منشن‌کردنی ڕۆڵ بە ID |

---

## گۆڕاوەکانی وەڵامی خۆکار (Auto Reply)

ئەم گۆڕاوانە دەتوانرێت لە بەشی وەڵامی خۆکاردا بەکاربهێنرێن:

| گۆڕاو | ڕوونکردنەوە |
| ------------- |:-------------:|
| `[user]` | منشن‌کردنی نێرەری پەیام |
| `[userName]` | ناوی نێرەر (بێ منشن) |
| `[invites]` | ژمارەی بانگهێشتەکان |
| `<@user_id>` | منشن‌کردنی ئەندام بە ID |
| `<#channel_id>` | منشن‌کردنی ژوور بە ID |
| `<@&role_id>` | منشن‌کردنی ڕۆڵ بە ID |

## گۆڕاوەکانی پەیامەکانی Embed

ئەم گۆڕاوانە دەتوانرێت لە بەشی پەیامەکانی Embed ـدا بەکاربهێنرێن:

| گۆڕاو | ڕوونکردنەوە |
| ------------- |:-------------:|
| `[text](link)` | بەستنی دەق بە لینک (کرتە بکەیت، لینکەکە دەکرێتەوە) |
| `<@user_id>` | منشن‌کردنی ئەندام بە ID |
| `<#channel_id>` | منشن‌کردنی ژوور بە ID |
| `<@&role_id>` | منشن‌کردنی ڕۆڵ بە ID |

---

## گۆڕاوەکانی سیستەمی Level (XP)

ئەم گۆڕاوانە دەتوانرێت لە بەشی سیستەمی Level ـدا بەکاربهێنرێن:

| گۆڕاو | ڕوونکردنەوە |
| ------------- |:-------------:|
| `[level]` | ئاستی (Level) ئێستای ئەندام |
| `[user]` | منشن‌کردنی ئەندام |
| `[user.username]` | ناوی ئەندام (بێ منشن) |
| `[user.tag]` | تاگی ئەندام |
| `[oldLevel]` | ئاستی پێشووی ئەندام |
