# پێشەکی 

پرۆبۆت بۆتێکی فرە تایبەتمەندییە و توانایەکی بەرزی هەیە بۆ ڕێکخستن (تەواو دەستکاری دەکرێت)، تایبەتمەندییەکانی وەک بەخێرهاتن، لۆگ، فەرمانە گشتی و کارگێڕییەکان، ڕۆڵە وەرگیراوەکان (Self-roles)، ئاگادارکردنەوەکانی سۆشیاڵ میدیا و پاراستنی سێرڤەرەکەت دابین دەکات.

## دەربارەی دۆکیومێنتەکانمان

دۆکیومێنتەکانمان ڕوونکردنەوەی گشتگیر دەربارەی چۆنیەتی بەکارهێنان و ڕێکخستنی هەر تایبەتمەندییەکی پرۆبۆت دابین دەکەن، لەگەڵ پێشنیارەکان بۆ دۆزینەوە و چارەسەرکردنی هەر کێشەیەک کە ڕەنگە ڕووبدات.

## بەشەکان

بە ئاسانی بڕۆ بۆ بەشێکی دیاریکراو لە ڕێگەی کرتەکردن لەسەر ناونیشانەکەی لە لیستی لای چەپدا. هەر بەشێک لاپەڕەی ڕێکخستنی تایبەتی خۆی هەیە.

## فەرمانەکان 

هەموو فەرمانەکانی پرۆبۆت لاپەڕەیەکی تایبەتیان هەیە کە نموونە و زانیاری بەکارهێنانی تێدایە. بڕۆ بۆ [لاپەڕەی فەرمانەکان](https://probot.io/commands) بۆ دەستکەوتنی وردەکاری زیاتر.

## ڕێکخستن

ئەم بەڵگانە سیستەمی ڕەنگی کۆدەکانی خوارەوە بەکاردەهێنن:

:::tip
ئاماژەیە بۆ ئامۆژگارییەکان پەیوەندیدار بە تایبەتمەندییەکان.
:::

:::info
زانیاری پەیوەندیدار بە تایبەتمەندییەکان دابین دەکات.
:::

:::caution
ئاگادارییەکان پەیوەندیدار بە تایبەتمەندییەکان دەردەخات.
:::

:::note
تێبینییەکان پەیوەندیدار بە تایبەتمەندییەکان پیشان دەدات.
:::

:::danger
ئاماژە دەکات بۆ هەڵەکان، کێشەکان و شتە ناسراوەکان کە دەکرێت دوور بخەیتەوە، ئەوانەی کە دەکرێت کاریگەری خراپ هەبێت لەسەر پرۆسەی کار.
:::
