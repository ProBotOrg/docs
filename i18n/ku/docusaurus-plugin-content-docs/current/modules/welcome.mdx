# ڕێکخستنی نامەی پێشوازیکردن و ماڵئاوایی لە سێرڤەرەکەت

نامەی **پێشوازیکردن** و **ماڵئاوایی** ڕۆڵێکی گرنگیان هەیە لە دروستکردنی  
کۆمەڵگایەکی دوستانە و جوان لە سێرڤەرەکەت.  
نامەی پێشوازیکردن دەتوانێت ئەندامی نوێ هەست بە پێزانین بکات  
و نامەی ماڵئاوایی دەتوانێت کاتێک ئەندامێک سێرڤەر جێدەهێڵێت  
هێشتا کاریگەری ئەرێنی جێبهێڵێت.

ئەم بەڵگەنامەیە تێڕوانینێکی گشتی پێشکەش دەکات  
بۆ هەنگاوە پێویستەکان بۆ ڕێکخستنی نامەی پێشوازیکردن و ماڵئاوایی  
لە سێرڤەرەکەت لە ڕێگەی **پرۆبۆت**.

---

:::info
بۆ ڕێکخستنی نامەی پێشوازیکردن و ماڵئاوایی، ئەم هەنگاوانە جێبەجێ بکە:

1. بچۆ بۆ [داشبۆرد](https://probot.io/ku/dashboard) و چوونەژوور بکە.
2. سێرڤەرەکەت لە لیستەی لای ڕاست هەڵبژێرە.
3. بچۆ بۆ بەشی پێشوازیکردن و ماڵئاوایی و دڵنیابە لە چالاکبوونی.
:::

---

## ڕێکخستنی نامەی پێشوازیکردن

### ناردنی نامە

نامەی پێشوازیکردن شێوەیەکی باشە بۆ پێشوازیکردنی ئەندامانی نوێ  
و هەست بە پێزانین پێیان بدات.

بۆ ڕێکخستن:

1. هەڵبژاردەی **نامەی پێشوازیکردن** چالاک بکە لە داشبۆرد.
2. ئەو دەقە بنووسە کە دەتەوێت بۆ ئەندامانی نوێ بنێردرێت.
3. دیاری بکە نامەکە:
   - نێردرێت بۆ **نامەی تایبەتی (DM)**  
   - یان بۆ **ژوورێکی دیاریکراو** لە سێرڤەر

---

### ناردنی وێنە

ناردنی **وێنەی پێشوازیکردن** ڕێگایەکی جوانی ترە  
بۆ دروستکردنی کۆمەڵگایەکی پێشوازیکەر.

بۆ ڕێکخستن:

1. هەڵبژاردەی ناردنی وێنەی پێشوازیکردن چالاک بکە.
2. یەکێک لەم سێ هەڵبژاردەیە هەڵبژێرە:
   - لەگەڵ نامەی پێشوازیکردن
   - پێش نامەی پێشوازیکردن
   - بۆ ژوورێکی دیاریکراو

#### ڕێکخستنی وێنەی پێشوازیکردن

- **پاشبنەما**: هەڵبژاردنی پاشبنەمای شەفاف یان وێنەی تایبەت  
- **ئافتار**: هەڵبژاردنی شێوەی چوارگۆشە یان بازنەیی، لەگەڵ قەبارە و شوێن  
- **ناوی بەکارهێنەر**: دیاریکردنی شوێن، قەبارە، ڕەنگ و ئاراستەی دەق  
- **دەق**: نووسینی دەقی دڵخواز و دیاریکردنی شوێن و ڕەنگی

:::caution
قەبارەی وێنەی پێشوازیکردن پێویستە کەمتر بێت لە **1MB**  
و فۆرماتە پشتیوانیکراوەکان ئەمانەن: `JPG`، `PNG`، `JPEG`
:::

---

## ڕێکخستنی نامەی ماڵئاوایی

نامەی ماڵئاوایی ڕێگایەکی باشە بۆ نیشاندانی ڕێز  
بۆ ئەندامانی ئەوەی سێرڤەر جێدەهێڵن.

بۆ ڕێکخستن:

1. هەڵبژاردەی **نامەی ماڵئاوایی** چالاک بکە.
2. ئەو دەقە بنووسە کە دەتەوێت لە کاتی دەرچوونی ئەندام بنێردرێت.
3. ژوورەکە دیاری بکە کە نامەی ماڵئاوایی تێدا بنێردرێت.

---

## زانیاری زیاتر

:::tip
- دەتوانیت **گۆڕاوەکان (Variables)** وەک ناوی ئەندام یان ناوی سێرڤەر  
  بەکاربهێنیت بۆ دروستکردنی نامەی پێشوازیکردن و ماڵئاوایی تایبەت.  
  بۆ نموونە سەردانی  
  [گۆڕاوەکانی پێشوازیکردن](https://docs.probot.io/ku/docs/getting-started/variables#%D9%85%D8%AA%D8%BA%D9%8A%D8%B1%D8%A7%D8%AA-%D8%A7%D9%84%D8%AA%D8%B1%D8%AD%D9%8A%D8%A8)
:::

:::info
- ڕێکخستنی باشی نامە و وێنەی پێشوازیکردن و ماڵئاوایی  
  یارمەتیدەرە بۆ دروستکردنی سێرڤەرێکی جیاواز و یادگار.  
  هەوڵبدە چەند شێوازێک تاقی بکەیت  
  تا باشترین هەڵبژاردە بۆ کۆمەڵگاکەت بدۆزیتەوە.
:::
