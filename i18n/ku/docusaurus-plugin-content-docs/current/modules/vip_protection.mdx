# پاراستنی تایبەت: پاراستنی پرۆبۆت بۆ ئاسایشکردنی سێرڤەرەکەت

:::caution
پاراستنی تایبەت تەنها تایبەتمەندی **پرێمیۆم** ـە.  
ئەگەر ئیشتراکی پرێمیۆم ـت نییە، ناتوانیت ئەم پاراستنە ڕێکبخەیت.
:::

**پرۆبۆت** یەکێکە لە باشترین بۆتەکان لە دیسکۆرد،  
یارمەتیت دەدات لە بەڕێوەبردنی سێرڤەر و پێشکەشکردنی تایبەتمەندییە جیاوازەکان.  
یەکێک لە گرنگترین ئەم تایبەتمەندییانە **پاراستنی تایبەت** ـە  
کە ئاسایشی سێرڤەر دابین دەکات بە سنووردارکردنی کردارەکانی بەڕێوەبەران  
و دانانی سنوور بۆ ئەنجامدانی کردارە بەڕێوەبەرییەکان.

---

:::info
بۆ ڕێکخستنی پاراستنی تایبەت لە داشبۆرد، ئەم هەنگاوانە جێبەجێ بکە:

1. بچۆ بۆ [داشبۆرد](https://probot.io/ku/dashboard) و چوونەژوور بکە.
2. سێرڤەرەکەت لە لیستەی لای ڕاست هەڵبژێرە.
3. بچۆ بۆ بەشی **پاراستنی تایبەت** و دڵنیابە لە چالاکبوونی.
:::

---

## زیادکردنی ڕۆڵێکی دیاریکراو

لە دۆخی بنەڕەتدا، سنوورەکان بۆ ڕۆڵی `@everyone` بێ‌سنوورن.  
**پرۆبۆت** ڕێگەت پێدەدات ڕۆڵی دیکە زیاد بکەیت  
بۆ ئەوەی سنوور و قەید بۆیان دابنێیت.

---

## سنوورەکانی مۆڵەت (Permission Limits)

بە بەکارهێنانی پاراستنی تایبەت،  
دەتوانیت کردارەکانی بەڕێوەبەران سنووردار بکەیت  
و ڕێگری لە بەکارهێنانی زیاتر بکەیت کە لەوانەیە ببێتە هۆی کێشە.

سنوورەکان بۆ ئەم کردارە بەڕێوەبەرییانە دابنرێن:

- قەدەغەکردن (Ban)
- دەرکردن (Kick)
- ژوورەکان (دروستکردن / دەستکاری / سڕینەوە)
- ڕۆڵەکان (دروستکردن / دەستکاری / سڕینەوە)

:::note
سنوورەکانی ڕۆڵ پەیوەندی بە **نوێکردنەوەی ڕۆڵی ئەندام** نییە.
:::

---

## کردارەکان (Actions)

پاراستنی تایبەت سێ جۆری کردار پێشکەش دەکات بۆ هەموو سنوورەکان:

1. سڕینەوەی هەموو مۆڵەتە بەڕێوەبەرییەکان لە ڕۆڵەکان.
2. سڕینەوەی هەموو مۆڵەتە بەڕێوەبەرییەکان لە ڕۆڵەکانی ئەندام (**پێشنیارکراو**).
3. لابردنی ڕۆڵە بەڕێوەبەرییەکان لە ئەندام.

دەتوانیت ئەو کردارە هەڵبژێریت کە گونجاوە بۆ سێرڤەرەکەت.

:::caution
کاتێک یەکێک لە دوو هەڵبژاردەی یەکەم هەڵدەبژێریت،  
خاوەنی سێرڤەر **کۆدی گەڕاندنەوە** وەردەگرێت  
بۆ گەڕاندنەوەی ئەو مۆڵەتانەی پرۆبۆت سڕیوەتەوە.
:::

---

## سنوورەکانی فەرمانەکان

پاراستنی تایبەت هەروەها سنوور بۆ فەرمانە بەڕێوەبەرییەکان دابین دەکات  
کە لەلایەن **پرۆبۆت** بەکاردەهێنرێن.

دەتوانیت سنوور بۆ ئەم فەرمانانە دابنێیت:

- /mute
- /mute voice
- /timeout
- /kick
- /ban
- /role

### تایبەتمەندی فەرمانی `/role`

دوو هەڵبژاردە هەیە:

- **ڕۆڵە ناچالاکەکان**:  
  ڕێگری لەوە بکە کە ئەندامان بە فەرمانی `/role`  
  ڕۆڵی دیاریکراو بدەن.

- **ڕۆڵە چالاکەکان**:  
  ڕێگەدان تەنها بۆ دانانی ڕۆڵە دیاریکراوەکان  
  بە فەرمانی `/role`.

:::caution
ئەندامانی خاوەن مۆڵەتی `Administrator`  
لە سنوورەکانی ڕۆڵ دەرچوو دەبن.
:::

:::tip
ئەگەر ئەندامێک چەند ڕۆڵێکی هەبێت،  
سنووری بەرزتر جێبەجێ دەکرێت.  
ڕۆڵی `@everyone` ڕۆڵی بنەڕەتی هەموو ئەندامانە.
:::

:::warning
لە هەندێک حاڵەتدا ئەم سنوورانە لەسەر بۆتەکان جێبەجێ نابن  
بەهۆی شێوازی کارکردنیان.  
لەوانەیە ئاگادار بیت لە بانگهێشتکردنی بۆتە نەناسراوەکان.
:::

---

## سنوورەکان (Limits)

هیچ سنوورێکی دیاریکراو نییە بۆ ژمارەی سنوورەکان.  
بێ تردید سنوورەکان بەپێی پێویستی خۆت دابنێ.

---

## ڕێکخستنەکانی پاراستن

پاراستنی تایبەت هەڵبژاردەی زیاتر دابین دەکات  
بۆ دڵنیابوون لە ئاسایشی سێرڤەرەکەت:

### ڕۆڵەکان بێ مۆڵەتی بەڕێوەبەری

**پرۆبۆت** ڕێگری دەکات لەوەی ئەم ڕۆڵانە  
مۆڵەتی بەڕێوەبەری وەربگرن  
بە سڕینەوەی مۆڵەتەکان کاتێک پێیان دەدرێت.

:::tip
پێشنیار دەکرێت ڕۆڵی `@everyone`  
و **ڕۆڵە خۆکارەکان** زیاد بکەیت بۆ ئەم لیستە.
:::

---

### کردار کاتێک بۆتێک ڕۆڵی بەڕێوەبەری وەربگرێت

**پرۆبۆت** دەتوانێت تێبگات کاتێک بۆتێکی تر  
ڕۆڵی بەڕێوەبەری پێدەدرێت و یەکێک لەم کردارە ئەنجام بدات:

1. سڕینەوەی هەموو ڕۆڵە بەڕێوەبەرییەکان لە بۆت (**پێشنیارکراو**).
2. سڕینەوەی هەموو مۆڵەتە بەڕێوەبەرییەکان لە ڕۆڵەکان (پرێمیم تایر 2).
3. دەرکردنی بۆت.

دوای هەڵبژاردنی کردار، هەڵبژاردەیەکی نوێ دەردەکەوێت بە ناوی  
**بۆتە ڕێگەپێدراوەکان**،  
لەوێ دەتوانیت ID ـی بۆتە ئاسایشەکان بنووسیت  
بۆ ئەوەی لە پاراستن دەرچوو بن.

---

### کێ دەتوانێت بچێتە داشبۆردی پرۆبۆت؟

لە دۆخی بنەڕەتدا،  
هەر ئەندامێک کە مۆڵەتی `Administrator` ـی هەبێت  
دەتوانێت بچێتە داشبۆردی سێرڤەر.

دەتوانیت دەستگەیشتن سنووردار بکەیت بۆ هەڵبژاردەکان:

1. هەر ئەندامێک بە مۆڵەتی `Administrator`
2. تەنها خاوەنی سێرڤەر + ئەندامانی دیاریکراوی خاوەن `Administrator`

کاتێک هەڵبژاردەی دووەم هەڵدەبژێریت،  
خانەی **ئەندامان** دەردەکەوێت  
کە لێیدا دەتوانیت ID ـی ئەو ئەندامانە بنووسیت  
کە دەتەوێت ڕێگەیان پێبدەیت بچنە داشبۆرد.

تێبینی: ئەم ئەندامانە پێویستە هەروەها مۆڵەتی `Administrator` ـیان هەبێت.

### سنووردارکردنی دەستکاری پاراستنی تایبەت

دەتوانیت دەستکاری تایبەتمەندییەکانی پاراستنی تایبەت  
تەنها بۆ **خاوەنی سێرڤەر** سنووردار بکەیت.
