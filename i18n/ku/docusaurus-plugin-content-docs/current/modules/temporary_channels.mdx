# ژوورە کاتییەکان: پرۆبۆت ژوورێکی دەنگی کاتی بۆ بەکارهێنەر دروست دەکات

تایبەتمەندی **ژوورە کاتییەکان** لە **پرۆبۆت** ڕێگە دەدات ئەندامان  
ژوورە دەنگییە کاتییەکان لە سێرڤەر دروست بکەن،  
بەڕێوەی ببرن و بەپێی ئارەزووی خۆیان دەستکارییان بکەن.

لەو بەڵگەنامەیەدا فێر دەبیت چۆن ئەم تایبەتمەندییە  
بە شێوەیەکی دروست ڕێکبخەیت لە سێرڤەرەکەت،  
لەوانەش دروستکردنی کاتێگۆری و ڕێکخستنی هەڵبژاردەکان بۆ ئەندامان.

---

:::info
بۆ ڕێکخستنی ژوورە کاتییەکان لە داشبۆرد، ئەم هەنگاوانە جێبەجێ بکە:

1. بچۆ بۆ [داشبۆرد](https://probot.io/ku/dashboard) و چوونەژوور بکە.
2. سێرڤەرەکەت لە لیستەی لای ڕاست هەڵبژێرە.
3. بچۆ بۆ بەشی **ژوورە کاتییەکان** و دڵنیابە لە چالاکبوونی.
:::

---

## ڕێکخستن (Setup)

### ڕێکخستنی کاتێگۆری

بۆ دەستپێکردن، پێویستە **کاتێگۆرییەک** دروست بکەیت  
بۆ ژوورە کاتییەکان، تا هەموو ژوورە دەنگییە کاتییەکان  
لە ناو ئەو کاتێگۆرییەدا دابنرێن.

---

### دروستکردنی ژوور

بۆ بەکارهێنانی ئەم تایبەتمەندییە،  
ئەندامان پێویستە بچنە ناو ژوورێکی دەنگی تایبەت.

کاتێک ئەندام بچێتە ناو ئەو ژوورە:
- **پرۆبۆت** ژوورێکی دەنگی تایبەت بۆی دروست دەکات
- ئەندام ڕاستەوخۆ گواسترێتەوە بۆ ژوورەکەی خۆی

---

:::info
دەتوانیت ئەم هەڵبژاردانە بەپێی پێویستی سێرڤەرەکەت ڕێکبخەیت:

- **زیادکردنی کات بۆ سڕینەوەی ژوورە کاتییەکان**  
  دیاری بکە ژووری دەنگی دوای چەند کاتێک بسڕدرێتەوە  
  کاتێک بەتاڵ دەبێت.

- **سنووری ژوورەکان بۆ هەر ئەندامێک**  
  ڕێگری لە سپامی ژوورەکان بکە  
  بە دیاریکردنی زۆرترین ژمارەی ژوورە کاتییەکان  
  کە ئەندامێک دەتوانێت لە یەک کاتدا دروست بکات.

- **زۆرترین ژمارەی بەکارهێنەران بۆ ژوور**  
  دیاری بکە چەند بەکارهێنەر دەتوانن  
  لە یەک کاتدا بچنە ناو ژوورێکی کاتی.
:::

---

:::caution
ئەندامانی خاوەن مۆڵەتی `Move Members`  
دەتوانن سنووری ژوورەکان تێپەڕ بکەن.
:::

---

بە ڕێکخستنی ئەم هەڵبژاردانە بە شێوەی دروست،  
ئەندامان دەتوانن ژوورە دەنگییە تایبەتییەکانی خۆیان  
دروست بکەن، بەڕێوەی ببرن و بە ئاسانی کۆنترۆڵیان بکەن 🎧
