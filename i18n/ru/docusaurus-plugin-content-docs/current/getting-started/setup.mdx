# Настройка ProBot

:::info

Для того, чтобы пригласить ProBot на ваш сервер, необходимы определенные разрешения. Без них бот не может быть добавлен на ваш сервер. К этим разрешениям относятся:

- Право `Администратор`
- Право `Управление сервером`

:::

## Шаг 1: Приглашение ProBot

Воспользуйтесь [инвайт-ссылкой](https://probot.io/invite) для того, чтобы пригласить ProBot на ваш сервер и авторизуйте ваш аккаунт, выбрав свой сервер

## Шаг 2: Авторизация и предоставление разрешений

Во время приглашения ProBot вам будет предложено авторизовать аккаунт и выбрать необходимые разрешения для правильной работы бота

## Шаг 3: Настройка роли ProBot

Для того, чтобы увидеть роли сервера в некоторых модулях панели управления, переместите роль ProBot выше остальных ролей. Для этого:
- Перейдите в настройки сервера
- Выберите Роли
- Зажмите и переместите роль ProBot выше всех

:::caution

Discord не всегда отображает роли в панели управления, даже когда роль ProBot выше всех. Если это происходит, попробуйте эти шаги:
1. Опустите роль ProBot
2. Сохраните изменения
3. Поднимите роль ProBot
4. Снова сохраните изменения

:::

## Нужна помощь?

:::info

Если вам нужна дополнительная помощь в настройке ProBot, присоединитесь к [Серверу поддержки](https://discord.gg/probot) и попросите о помощи. Наша команда будет рада помочь вам!

:::
