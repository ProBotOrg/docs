# Команды

В этом разделе вы лучше поймёте команды ProBot, их применение и примеры использования

:::caution

- Так как слеш является префиксом бота по умолчанию, все примеры использования будут на примере слеш-команд `/`

- Знаки `[` `]` обязательны для каждой команды, на которую ссылается этот знак, а `(` `)` необязательны

:::

## Общее 

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /credits | Показать ваши или чьи-либо кредиты | /credits [пользователь] (число) |
| /rep | Наградить кого-нибудь очком репутации. Можно использовать раз в 24 часа | /rep [юзер] |
| /moveme | Перемещает вас в другой голосовой канал | /moveme [канал или юзер] |
| /color | Сменить ваш цвет на сервере | /color [номер цвета] |
| /colors | Показывает список доступных цветов | /colors |
| /short | Сокращает URL | /short [URL] |
| /roll | Бросок игральных костей | /roll (число) |

## Система уровней

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /profile | Посмотреть ваш или чей-либо кастомизируемый персональный общедоступный профиль | /profile (юзер) |
| /rank | Посмотреть вашу или чью-либо рейтинговую карточку на сервере | /rank (юзер) |
| /top | Отобразить топ участников по тексту или войсу | /top (day (день)/week (неделя)/month (месяц)) |
| /title | Меняет заголовок вашего профиля | /title [новый заголовок] |
| /setxp | Устанавливает опыт пользователя | /setxp [юзер] [тип] [новый_опыт] |
| /setlevel | Устанавливает уровень юзера | /setlevel [юзер] [тип] [новый_уровень] |

## Информация

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /user | Показывает информацию о себе или пользователе, например ID и дату вступления | /user (юзер) |
| /avatar | Получить аватар пользователя | /avatar (юзер) |
| /server | Показывает информацию о сервере | /server |
| /roles | Получить список ролей сервера и число участников | /roles |

## Модерация

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /setnick | Меняет никнейм участника | /setnick [юзер] (никнейм) |
| /ban | Банит участника | /ban [юзер] (время) (причина) |
| /unban | Снимает бан с участника | /unban [юзер или его ID] |
| /kick | Кикает участника | /kick [юзер] (причина) |
| /vkick | Кикает участника с голосового канала | /vkick [юзер] |
| /mute text | Замутить участника, вследствие чего он не сможет печатать в текстовых каналах | /mute text [юзер] (время) (причина) |
| /unmute text | Снимает мут с участника | /unmute [юзер] |
| /mute voice | Замутить участника, вследствие чего он не сможет говорить в голосовых каналах | /mute voice [юзер] (время) (причина) |
| /unmute voice | Снимает голосовой мут с участника | /unmute voice [юзер] |
| /timeout | Отправить участника в тайм-аут | /timeout [юзер] (время) (причина) |
| /untimeout | Убирает тайм-аут с участника | /untiemout [юзер] |
| /clear | Очищает сообщения канала | /clear (юзер/боты) (количество сообщений) | 
| /move | Перемещает участника в другой голосовой канал | /move [юзер] (юзер или канал) |
| /role give | Добавить/снять роль(и) с участника | /role give [юзер] [роль] (bulk) |
| /role remove | Снять роль(и) с участника | /role remove [юзер] [роль] (bulk) |
| /role multiple | Добавить/снять роль(и) с участников | /role multiple [give(добавить) или remove (снять)] [роль] [тип_выбора] (необходимая роль) |
| /points set | Установить баллы пользователя | /points set [юзер] [баллы] |
| /points list | Посмотреть все баллы | /points list (роль) (страница) |
| /points decrease | Уменьшить баллы пользователя | /points decrease [юзер] [баллы] |
| /points increase | Добавить баллы пользователя | /points increase [юзер] [баллы] |
| /points reset | Сбросить баллы | /points reset (юзер) |
| /warn | Предупреждает участника | /warn [юзер] [причина] |
| /warn_remove | Снять варны сервера или пользователя | /warn_remove [юзер/all (все)/ID варна] |
| /warnings | Получить список варнов сервера или пользователя | /warnings (юзер) |
| /lock | Отключает `@everyone` отправку сообщений в выбранном канале | /lock (канал) (причина) |
| /unlock | Разрешает `@everyone` отправлять сообщения в выбранном канале | /unlock (канал) |
| /setcolor | Изменяет цвет роли по hex-коду | /setcolor [имя роли] [код цвета]
| /slowmode | Включить или выключить медленный режим в канале | /slowmode (время) |
| /reset | Сбросить опыт (текст/войс/приглашения) для всех или указанных участников | /reset [text (текст) / voice (войс) / invite (приглашения) / limits (лимиты)] [all (все) / юзер] |

<details>

<summary>Важные примечания по модерации</summary>

- Только пользователи с разрешениями `Администратор`, `Роли модераторов` в панели управления, или с разрешением на эту команду могут использовать команды модерации. __Например__, пользователи с разрешением на бан могут использовать команду `/ban`

- Если время не указано, то время по умолчанию для всех команд с указанием времени — **3 месяца**; это может быть изменено в любое время

</details>

## VIP

:::caution

Следующие команды доступны только премиум-подписчикам

:::

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /vip info | Показывает информацию о вашем премиум-боте | /vip |
| /vip move | Переместить вашего премиум-бота на новый сервер `Уровень 2` | /vip move [номер] |
| /vip restart | Перезапустить вашего премиум-бота `Уровень 2` | /vip restart |
| /vip whitelist | Добавить пользователя в белый список для доступа к панели управления ProBot | /vip whitelist [пользователь] |
| /vip avatar | Сменить аватар вашему боту `Уровень 2` | /vip avatar |
| /vip username | Сменить ник вашему боту `Уровень 2` | /vip username [имя] |
| /vip status | Сменить статус вашему боту `Уровень 2` | /vip status [статус] |
| /vip transfer | Передать владение ботом другому пользователю `Уровень 2` | /vip transfer [пользователь]
