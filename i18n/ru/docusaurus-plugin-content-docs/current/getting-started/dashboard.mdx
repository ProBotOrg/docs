# Панель управления: ваш универсальный магазин для управления сервером

Панель управления является сердцем конфигурации вашего сервера. Она предоставляет простой в использовании интерфейс для управления всеми модулями и командами. Попрощайтесь со сложными и занимающими время текстовыми командами!

## Доступ к Панели управления

Доступ к Панели управления прост и понятен. Вот шаги:
- Перейдите по [ссылке](https://probot.io/) и войдите в свой аккаунт
- Кнопка панели управления появится на этой же странице, кликните по ней для доступа к панели управления

:::caution
Вам необходимо право `Администратора` для того, чтобы увидеть ваш сервер в списке
:::

## Настройка Модулей и Команд

Каждый модуль имеет кнопку переключения в правом верхнем углу, которая позволяет вам отключить все команды, в том числе сам модуль. Кроме того, каждая команда имеет свою кнопку переключения для её включения или выключения, а также кнопку настройки для внесения изменений

Вот некоторые опции настройки, доступные для каждой команды:
- **Псевдоним**: Кастомный псевдоним для команды делает её проще в запоминании и использовании
- **Разрешённые роли**: Ограничить круг лиц, которые могут использовать команду определёнными ролями
- **Отключённые роли**: Ограничить определённым ролям использование команды
- **Разрешённые каналы**: Бот будет отвечать на команду только в указанных каналах
- **Отключённые каналы**: Бот не будет отвечать на команду в этих каналах
- **Авто-удаление с удалением сообщения**: Ответ бота будет удалён, если отправитель удалит своё сообщение
- **Авто-удаление сообщения вызова команды**: Сообщение команды будет немедленно удалено после отправки
- **Авто-удаление сообщения ответа бота после 5 секунд**: Ответ бота будет удалён через 5 секунд после его отправки

:::caution
Не видите ваш сервер в списке? Это значит, что у вас нет права `Администратора` для полного доступа к Панели управления. Убедитесь, что ваша роль имеет право `Администратора`, выданное вашей роли в настройках
:::
