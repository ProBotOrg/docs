# Временные ссылки

Функция команды Вре́менной ссылки позволяет администраторам сервера установить команду для пользователей для приглашения друзей с лёгкостью. Следующие инструкции содержат пошаговое руководство по настройке команды.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите к панели управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

:::caution

Эта функция доступна только для премиум-подписчиков.

:::

## Название команды

Первым шагом является ввод имени команды, которую пользователи могут ввести, чтобы получить персональную ссылку. Команда может быть любым желаемым текстом, префиксом, символом и буквой.

## Сообщения

### Сообщение, когда ссылка была запрошена

Когда пользователь вводит команду, сообщение появится как ответ, указывая на то, что ссылка была успешно отправлена в ЛС пользователю. Текст этого сообщения может быть изменён по желанию.

### Сообщение, когда ссылка не может быть отправлена

Если бот не может отправить ссылку, отобразится отдельное сообщение в ответ, указывая на то, что бот не смог отправить ссылку, потому что ЛС пользователя закрыт. Текст этого сообщения также может быть изменён по желанию.

## Роли

### Отключённые роли 

Администратор может предотвратить указанным ролям использование команды для ссылки.

### Разрешённые роли
 
Пользователи с указанными ролями смогут использовать команду для ссылки.

## Каналы

### Отключённые каналы

Бот не будет отвечать на команду ссылки в указанном канале.

### Разрешённые каналы

Бот ответит на команду ссылки в указанном канале.

## Информация о ссылке

### Сообщение текста со ссылкой

Сообщение, отправляемое в ЛС пользователю, когда он использовал команду, включая инвайт-ссылку с информацией об использовании ссылки и временем истечения. Текст этого сообщения может быть изменён по желанию.

### Использования ссылки

Администратор может указать, сколько раз можно использовать ссылку и лимит пользователей, которые могут присоединиться с этой ссылкой. Для установки количества использований ссылки предусмотрена полоса прокрутки.

### Канал приглашения по умолчанию

Администратор может указать, какой канал будет принадлежать к инвайт-ссылке. Например, если администратор установил канал `Тест`, пользователи, воспользовавшиеся ссылкой, будут перенаправлены на канал `Тест` после присоединения.

### Срок действия ссылки в минутах

Администратор может указать срок действия ссылки в минутах. Ссылка истечёт, если она использована за это время. Для установки срока действия ссылки предусмотрена полоса прокрутки.

## Время ожидания

### Время ожидания в часах

Администратор может установить время ожидания для пользователей на новый запрос ссылки.

### Сколько ссылок можно запросить до времени ожидания

Администратор может установить количество ссылок, которое можно запросить до времени ожидания.

### Сообщение времени ожидания

Когда пользователь попытается запросить новую ссылку перед тем, как время ожидания истечёт, появится сообщение в ответ, указывая на то, что он должен подождать указанное количество времени перед запросом новой ссылки. Текст этого сообщения может быть изменён по желанию.

## Отключить время ожидания

Администратор может разрешить пользователям с разрешением `Создание приглашения` обходить время ожидания и создавать ссылки в любое время.

## Удалить сообщение команды и ответ бота

Администратор может выбрать, удалит ли бот команду пользователя и ответ бота через пять секунд или сохранит их. Включение опции приведет к удалению сообщения, а выключение — к их сохранению.

:::important

Вот ограничения для наших вариантов вре́менной ссылки:

- Срок ссылки в минутах — `1440`
- Время ожидания в часах — `9999999999999`
- Сколько ссылок может быть запрошено перед временем ожидания — `9999999999999`

:::
