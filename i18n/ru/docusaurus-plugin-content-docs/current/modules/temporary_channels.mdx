# Временные каналы: Функция ProBot для каналов, управляемых пользователем

Вре́менные каналы — отличная функция в ProBot, которая позволяет пользователям создавать и управлять собственными временными каналами на сервере. С этой особенностью пользователи могут создавать и кастомизировать собственные каналы, как они хотят их видеть, с набором кастомных опций, которые владельцы серверов могут настраивать в соответствии со своими потребностями. В этом руководстве мы рассмотрим шаги по настройке временных каналов на вашем сервере ProBot, включая создание категории и настройку опций, доступных пользователям.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите к панели управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Настройка

### Установка категории

Чтобы приступить, вам нужно создать специальную категорию, где будут находиться все вре́менные каналы.

### Создать канал

Чтобы использовать эту функцию, участники сначала должны присоединиться к голосовому каналу, поэтому вы должны создать голосовой канал. Как только они в этом канале, они создают свой собственный временный канал.

:::info

Вы можете настроить следующие опции для удовлетворения потребностей вашего сервера:

- **Время до удаления канала**: Это количество времени, в течение которого временный канал может оставаться пустым, прежде чем он будет автоматически удалён. Этот параметр является настраиваемым и может быть установлен на любое значение, которое вы выберете.

- **Максимум каналов на пользователя**: Это максимальное количество вре́менных каналов, которые пользователь может создать одновременно. Эта опция ставится для предотвращения спама каналами, и настраивается так, что вы можете установить максимальное количество каналов, которое вам удобно.

- **Максимум пользователей на канал**: Это максимальное количество пользователей, которые могут присоединиться к вре́менному каналу одновременно. Эта опция настраиваемая, так что вы сможете установить максимальное количество пользователей, которое вам удобно.

:::

:::caution

Пользователи с правом `Перемещать участников` обходят лимит пользователей.

:::

С помощью этих настроек ваши пользователи смогут создавать свои собственные вре́менные каналы с желаемыми параметрами, а вы сможете легко управлять ими.
