# Автомодерация

Обеспечьте бесперебойную работу сервера без стресса с помощью Автомода ProBot. Попрощайтесь со спамерами и наслаждайтесь спокойствием, зная, что ProBot неустанно работает над поддержанием безопасной и надёжной среды для вашего сообщества

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями

1. Перейдите в панель управления и авторизуйтесь
2. Выберите ваш сервер в левом меню
3. Выберите модуль и убедитесь, что он включён

:::

## Опции обнаружения

Для обнаружения потенциального спама или вредоносных сообщений на вашем сервере доступны следующие опции:

:::info

У нас есть некоторые опции обнаружения, которые управляются правилом Автомода в Discord, которые будут указаны рядом с опцией Автомод. ProBot будет управлять теми действиями, которые не имеют упомянутого сканирования автомода в Discord

:::

> **СПАМ (5 СООБЩЕНИЙ/5 СЕК)**: Эта опция определяет, когда пользователь отправляет более 5 сообщений за 5-секундный промежуток

> **ПЛОХИЕ СЛОВА**: Эта опция определяет, содержит ли сообщение плохие слова **(Сканирование сообщений основано на Автомоде Discord)**

> **ДУБЛИРОВАННЫЙ ТЕКСТ**: Эта опция определяет сообщения, содержащие дублированный текст

> **ПОВТОРЯЮЩИЕСЯ СООБЩЕНИЯ**: Эта опция определяет сообщения, которые повторяют текст или строки, например "СПАМ СПАМ СПАМ СПАМ" **(Уровень 2)**

> **DISCORD-ПРИГЛАШЕНИЯ**: Эта опция определяет сообщения, содержащие инвайт-ссылки Discord **(Сканирование сообщений основано на Автомоде Discord)**

> **ССЫЛКИ**: Эта опция определяет сообщения, содержащие ссылки ** (Сканирование сообщений основано на Автомоде Discord)**

> **СПАМ КАПСОМ (70% > КАПС)**: Эта опция определяет сообщения, которые написаны капсом, например "ПРИВЕТ ЭТО PROBOT" **(Сканирование сообщений основано на Автомоде Discord)**

> **СПАМ EMOJI**: Эта опция определяет сообщения, содержащие 8 или более эмодзи

> **МАССОВОЕ УПОМИНАНИЕ**: Эта опция определяет сообщения, упоминающие большое количество пользователей **(Сканирование сообщений основано на Автомоде Discord)**

## Варианты действий

Каждый вариант обнаружения предлагает три различных варианта действий для удовлетворения конкретных потребностей вашего сообщества. К этим действиям относятся:

1. **Блокировать сообщение**: Удаляет сообщение, активировавшее обнаружение

2. **Замутить участника**: Мутит участника, отправившего запретное сообщение

3. **Отправить участника в тайм-аут**: Выдать тайм-аут участнику, который отправил запретное сообщение **(Доступны только опции обнаружения, которые управляются Правилом Автомода Discord)**

## Настройки

Каждый вариант обнаружения имеет опцию **Настройки**, обозначенную значком шестеренки `⚙️`. Нажатие на эту опцию раскроет ряд параметров, которые вы можете настроить для оптимизации использования Автомода ProBot для вашего сервера

> **Отключённые каналы**: Каналы, которые будут исключены из опций обнаружения

> **Отключённые роли**: Роли, которые будут исключены из опций обнаружения

В опциях автоматизации **Плохие слова** и **Ссылки** существуют дополнительные настройки, отсутствующие в опциях по умолчанию:

> **Плохие слова**

- Плохие слова: Список индивидуальных плохих слов, которые будут триггерить опции обнаружения

- НЕ ИСКАТЬ СОВПАДЕНИЕ, ЕСЛИ ЭТО ЧАСТЬ СЛОВА: Если включено, эта опция будет предотвращать обнаружение слов, которые содержат плохие слова, но не идентичны им

> **Ссылки**

- Разрешённые URL: Список ссылок, которые не будут триггерить опции обнаружения, даже если они содержат ссылки

- Запрещённые URL: Список ссылок, которые будут триггерить опции обнаружения, даже если они не содержат ссылок

## Дополнительные функции

Помимо описанных выше опций обнаружения и действий, Автомод ProBot также включает следующие функции:

> **Игнорируемые каналы**: Эти каналы будут исключены со всех опций обнаружения

> **Игнорируемые роли**: Эти роли будут исключены со всех опций обнаружения

> **Каналы только для изображений**: Позволяет вам отправлять только изображения в указанные каналы

> **Только YT-каналы**: Позволяет вам отправлять только YouTube-ссылки в указанные каналы
