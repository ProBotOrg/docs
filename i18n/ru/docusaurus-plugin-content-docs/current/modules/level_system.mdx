# Система уровней

С системой уровней ProBot активные пользователи будут автоматически награждены необычными уровневыми ролями, имеющими эксклюзивные права и доступ к тайным каналам, как только они достигнут указанного уровня!

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

Сначала у вас будет три различных варианта, каждый из которых можно настроить для получения более индивидуального уровневого опыта.

## Роли, не получающие XP

Используя это, вы можете запретить определённым пользователям с выбранными ролями накапливать очки опыта в любом месте вашего сервера.

## Каналы без получения XP

Вы можете запретить определённым каналам накапливать очки опыта.

## Канал повышения уровня

Отправляйте поздравительные сообщения пользователям, которые поднялись с одного уровня на более высокий. Вы также можете использовать уникальные переменные в этих сообщениях!

:::caution

**Сообщение о повышении уровня** будет отправлено только тем, кто повысил текстовый уровень. В настоящее время у нас нет сообщения о голосовом уровне.

:::

## Настройка

Процедура создания системы вознаграждения ролью описана ниже:


Появится кнопка `Добавить Награду Ролью`, где вы можете добавить награду ролью, свой уровень, которого должны достичь пользователи, и другие параметры, описанные ниже.

### Текстовый уровень

- Это текстовый уровень, которого должен достичь участник, чтобы получить роль за уровень, если она была установлена самостоятельно.

- Диапазон текстовых уровней — от 0 до 100; предоставляйте пользователям дополнительные вознаграждения по мере достижения ими необходимого уровня.

### Голосовой уровень

- Это голосовой уровень, которого должен достичь участник, чтобы получить роль за уровень, если она была установлена самостоятельно.

- Диапазон голосовых уровней — от 0 до 100; предоставляйте пользователям дополнительные вознаграждения по мере достижения ими необходимого уровня.

### Награды ролью

Награждайте участников особенными должностями по мере того, как они продвигаются по уровням и выполняют ваши требования.

:::caution

Если роли не отображаются, то роль бота находится ниже всех ролей сервера. Для решения этой проблемы переместите роль бота выше всех в иерархии сервера, чтобы убедиться, что роли отображаются.

:::

## Удалять после повышения уровня

Вы сами решаете, должны ли уровни складываться или удаляться, когда пользователи поднимаются на более высокие уровни.

- Если вы отключите опцию, роли будут складываться.

- Если вы включите опцию, роль будет удалена после достижения более высокого уровня.

## ЛС участнику

Уведомляет участника, когда он получил роль за текстовый уровень, через личные сообщения.

## Важные замечания

:::important Три потенциальные ситуации, которые могут возникнуть при установке требования к уровню

1. **И войс, и текст**: Если вы установите необходимость как текстового, так и голосового уровня, не оставляя `0` в любом параметре, то будет требоваться и текст, и голос. Это потребует от пользователя достижения обоих уровней для награждения ролью.

2. **Только текст**: Если вы установите необходимость уровня текста и оставите Голос на `0`, будет требоваться только Текст. Это потребует от пользователя достичь уровня, необходимого для Текста, чтобы получить награждение ролью.

3. **Только войс**: Если вы установите необходимость уровня голоса и оставите значение Текст, равным `0`, будет требоваться только Голос. Это потребует от пользователя достичь уровня, необходимого для Голоса, чтобы получить награждение ролью.

4. **Позиция роли**: Если вы выбрали одинаковые роли для награждения для многих уровней, не волнуйтесь — он все равно будет предоставлять пользователям указанную вами роль по мере достижения ими требований любого уровня, который вы установили для этой же роли.

:::

:::caution

Пользователям необходимо набирать текст в чатах, чтобы бот обновил данные об их XP и вознаградил их наградами голосового уровня, и только после этого бот сможет наградить их.

:::
