# Автороли

Функция Автороль ProBot позволяет автоматически назначать роли новым пользователям на их сервере Discord. В этом руководстве вы пройдете через шаги по установке и настройке этой функции.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Настройка

Получив доступ к модулю Автороли, вы можете начать настройку ролей, которые будут автоматически назначаться новым пользователям. Для этого воспользуйтесь меню выбора модуля, чтобы выбрать одну или несколько ролей по желанию.

## Премиум Автороль

Вы можете установить уникальную ссылку для авто- и определённых ролей, если вы премиум-подписчик.

1. Выберите `Добавить`
2. Теперь нажмите на кнопку `Настроить`.
3. Вставьте инвайт-ссылку
4. Выберите роли, которые вы хотите дать участникам, если они присоединятся по этой инвайт-ссылке.

:::caution

1. Роль бота должна быть выше выбранных авторолей.

2. Когда экран с приветствием Discord включён, новые пользователи должны принять правила, перед тем как они смогут получить автороль.

3. Бот не назначит роль, если роли пользователей имеют административные разрешения.

:::

Выполнив следующие шаги и приняв к сведению важные замечания, вы сможете успешно установить и настроить функцию Автороли на вашем сервере Discord с помощью ProBot.
