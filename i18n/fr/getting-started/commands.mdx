# Commandes

Dans cette section, vous obtiendrez une compréhension plus profonde des commandes de probot, de leur utilisation et d'un exemple de la façon de mettre de telles commandes en action.

:::important

- Comme les commandes de slash sont le préfixe de bot par défaut, tous les usages et exemples seront illustrés à la slash commande `/`.

- `[` `]` Des signes sont requis pour chaque commande référencée avec cet signe, mais `(` `)` sont facultatifs.

:::

## Général 

| Commande | Explication | Usage |
| ------------- |:-------------:|:-------------:|
| /credits | Montrez votre crédit ou celui de quelqu'un d'autre | /credits [utilisateur] (montant) |
| /rep | Attribuez à quelqu'un un point de réputation. Ne peut être utilisé qu'une fois toutes les 24 heures | /rep [utilisateur] |
| /moveme | Vous déplace vers un autre salon vocal | /moveme [salon ou utilisateur] |
| /color | Changez votre couleur sur le serveur | /color [numéro de couleur] |
| /colors | Vous répertorie toutes les couleurs disponibles | /colors |
| /short | Se raccourcit en URL | /short [URL] |
| /roll | Lancer des dés | /roll (numéro) |

<details>

<summary>Remarques Importantes Sur La Modération</summary>

- Seuls les utilisateurs disposant de l'autorisation `Administrateur`, `Rôles Modérateurs` sur le tableau de bord ou de l'autorisation pour cette commande peuvent utiliser les commandes de modération. __Par exemple__, les utilisateurs disposant d'une autorisation d'interdiction peuvent utiliser la commande "/ban".

- Si aucune heure n'a été spécifiée, la durée par défaut pour toutes les commandes d'entrée de temps est de **3 mois**; celui-ci peut être modifié à tout moment. Pour plus d'informations, veuillez visiter [modération] (). 

</details>
