# Configurer ProBot

:::info

Afin d'inviter ProBot sur votre serveur, certaines permissions sont requises. Sans ceux-ci, le bot ne peut pas être ajouté à votre serveur. Ces permissions sont:

- `Administrator` Permission 
- `Manage Servers` Permission

:::

## Étape 1: Inviter ProBot

Utilisez le [lien d'invitation](https://probot.io/invite) pour inviter ProBot sur votre serveur et autoriser votre compte, en sélectionnant votre serveur.

## Étape 2: Autoriser et Accorder des Autorisations

Après avoir invité ProBot, vous serez invité à autoriser votre compte et à sélectionner les permissions nécessaires pour que le bot fonctionne correctement.

## Étape 3: Configurer le Rôle de ProBot

Pour afficher les rôles de serveur dans certains modules de tableau de bord, déplacez le rôle de ProBot en haut de la hiérarchie des rôles de serveur. Pour faire ça:
- Allez aux paramètres du serveur
- Cliquez sur les rôles
- Maintenez-cliquez et faites glisser le rôle de ProBot en haut de la liste

:::caution

Discord n'affiche pas toujours les rôles sur le tableau de bord, même après avoir fait glisser le rôle de ProBot vers le haut. Si cela se produit, essayez ces étapes:
1. Faites glisser le rôle de ProBot vers le bas
2. Enregistrez les modifications
3. Faites glisser le rôle de ProBot vers le haut
4. Enregistrez à nouveau les modifications

:::

## Besoin d'aide?

Si vous avez besoin d'une assistance supplémentaire pour la configuration de ProBot, rejoignez le [serveur d'assistance](https://discord.gg/probot) et demandez de l'aide. Notre équipe se fera un plaisir de vous aider!

:::
