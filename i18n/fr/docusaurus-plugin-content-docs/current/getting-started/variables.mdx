# Variables du Tableau de Bord ProBot

Pour rendre vos messages plus professionnels et esthétiques, vous pouvez utiliser diverses variables pour y parvenir. Ce guide vous fournira toutes les variables que vous pouvez utiliser dans différentes sections de votre tableau de bord.

## Prérequis

Avant d'utiliser des variables, vous devez activer le mode développeur Discord sur votre appareil. Ce mode vous fournira l'ID dont vous avez besoin pour mentionner des choses spécifiques.

## Activer le Mode Développeur sur Discord

> Sur PC
1. Lancer Discord
2. Cliquer sur `User Settings`
3. Défilez jusqu'à `App Settings` — cliquez sur `Appearance`
4. Activer le Mode Développeur

> Sur Téléphone
1. Lancer Discord
2. Appuyez sur `User Settings`
3. Défilez jusqu'à `App Settings` — appuyez sur `Behavior`
4. Activer le Mode Développeur

Une fois que vous avez activé le mode développeur, vous êtes maintenant prêt à utiliser les variables.

## Variables de Bienvenue

**Les variables suivantes peuvent être utilisées dans la section Bienvenue de votre tableau de bord Discord:**

| Variable | Description | 
| ------------- |:-------------:|
| `[user]` | Cette variable mentionne le membre nouveau venu | 
| `[inviter]` | Cette variable mentionne l'invitant |
| `[server]` | Cette variable affiche le nom du serveur |
| `[userName]` | Cette variable affiche le nom du membre nouveau venu |
| `[inviterName]` | Cette variable affiche le nom de l'invitant |
| `[memberCount]` | Cette variable affiche le nombre de membres atteints |
| `<@user_id>` | Cette variable mentionne un utilisateur spécifique à l'aide de son ID |
| `<#channel_id>` | Cette variable mentionne un salon spécifique à l'aide de son ID |
| `<@&role_id>` | Cette variable mentionne un rôle spécifique à l'aide de son ID |

## Variables de Répondeur Automatique

**Les variables suivantes peuvent être utilisées dans la section Répondeur Automatique de votre tableau de bord Discord:**

| Variable | Description | 
| ------------- |:-------------:|
| `[user]` | Cette variable mentionne le membre qui a tapé le déclencheur |
| `[userName]` | Cette variable affiche le nom de l'utilisateur qui a tapé le déclencheur |
| `[invites]` | Cette variable affiche le nombre d'invitations que vous avez atteint |
| `<@user_id>` | Cette variable mentionne un utilisateur spécifique à l'aide de son ID |
| `<#channel_id>` | Cette variable mentionne un salon spécifique à l'aide de son ID |
| `<@&role_id>` | Cette variable mentionne un rôle spécifique à l'aide de son ID |

## Variables des Messages de Embed

**Les variables suivantes peuvent être utilisées dans la section Messages de Embed de votre tableau de bord Discord:**

| Variable | Description | 
| ------------- |:-------------:|
| `[text](link)` | Cette variable crée un lien hyperlien |
| `<@user_id>` | Cette variable mentionne un utilisateur spécifique à l'aide de son ID |
| `<#channel_id>` | Cette variable mentionne un salon spécifique à l'aide de son ID |
| `<@&role_id>` | Cette variable mentionne un rôle spécifique à l'aide de son ID |

## Système de Niveau 

**Les variables suivantes peuvent être utilisées dans la section Système de Niveau de votre tableau de bord Discord:**

| Variable | Description | 
| ------------- |:-------------:|
| `[level]` | pour imprimer le niveau donné | 
| `[user] `| envoyer un ping à l'utilisateur |
| `[user.username]` | pour imprimer le nom d'utilisateur |
| `[user.tag]` | pour apporter une balise d'utilisateur |
| `[oldLevel] ` | pour apporter l'ancien niveau d'utilisateur |

En utilisant les variables listées ci-dessus, vous pouvez rendre vos messages plus professionnels et spécial.
